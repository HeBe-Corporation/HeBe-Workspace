## 응답

# 미백 성분 3종 조합 (Glutathione + Niacinamide + Arbutin) 시너지 분석

## 각 성분의 작용 메커니즘

| 성분 | 작용점 | 메커니즘 |
|------|--------|----------|
| **Glutathione** | 티로시나제 억제 + 항산화 | 페오멜라닌(밝은색) 생성 유도, 유멜라닌(어두운색) 억제 |
| **Niacinamide** | 멜라노좀 전달 차단 | 멜라노사이트 → 케라티노사이트 이동 억제 |
| **Arbutin** | 티로시나제 억제 | 하이드로퀴논 유도체, 티로신→도파 변환 차단 |

## 시너지 효과

```
[멜라닌 생성 경로]

티로신 → 도파 → 도파퀴논 → 멜라닌 → 피부 표면
   ↑         ↑              ↑           ↑
 Arbutin   Glutathione    Glutathione  Niacinamide
 (차단)    (경로 전환)     (항산화)     (전달 차단)
```

**3중 블록 효과**:
1. Arbutin + Glutathione: 티로시나제 이중 억제
2. Glutathione: 생성된 멜라닌을 밝은 타입으로 전환
3. Niacinamide: 만들어진 멜라닌의 피부 표면 도달 차단

## 권장 배합 비율

### 일반 제형 (세럼/에센스)

| 성분 | 권장 농도 | 최대 농도 | 비고 |
|------|-----------|-----------|------|
| Glutathione | 0.5-2% | 5% | 안정성 이슈로 리포좀 권장 |
| Niacinamide | 4-5% | 10% | 5% 이상 효능 증가 미미 |
| Arbutin (α) | 1-2% | 2% | β-arbutin은 7%까지 가능 |

### 최적 조합 제안

**Option A - 데일리 케어용**
```
Niacinamide: 4%
α-Arbutin: 1%
Glutathione: 0.5%
```

**Option B - 집중 케어용**
```
Niacinamide: 5%
α-Arbutin: 2%
Glutathione: 2% (리포좀 캡슐화)
```

## 제형 시 주의사항

### pH 조건
- **최적 pH**: 5.0-6.5
- Arbutin: 약산성에서 안정
- Niacinamide: pH 6 이하에서 니아신 전환 위험 → 5.5-6.0 권장
- Glutathione: 산화 방지 위해 질소 충전 또는 리포좀화

### 배합 금기
| 조합 | 문제점 |
|------|--------|
| Niacinamide + 순수 비타민C | pH 차이로 니아신 전환, 홍조 유발 |
| Arbutin + 강산성 | 하이드로퀴논 분해 위험 |
| Glutathione + 금속이온 | 산화 촉진 |

### 안정화 전략
1. **Glutathione**: 리포좀 캡슐화 또는 환원형(GSH) 사용
2. **산화방지제 추가**: Vitamin E, Ferulic acid
3. **차광 용기**: 갈색/불투명 용기 필수

## 효능 극대화 팁

1. **아침**: Niacinamide + Arbutin (자외선 차단제와 병용)
2. **저녁**: Glutathione 집중 (항산화 + 재생 시간 활용)
3. **주 2-3회**: 3종 풀 조합 집중 케어
