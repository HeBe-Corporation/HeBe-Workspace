# 밀크 클렌저 제형 설계 가이드

## 1. 유화 안정성 확보 방법

### 유화 타입 선택
- **O/W (수중유형)**: 밀크 클렌저의 표준. 사용감 가볍고 세정력 적당
- **W/O (유중수형)**: 리치한 텍스처, 메이크업 제거력 높음

### 안정화 전략
| 방법 | 설명 |
|------|------|
| **적정 점도 확보** | 증점제(Carbomer, Xanthan Gum)로 입자 이동 억제 |
| **입자 크기 균일화** | 호모믹서 10,000rpm 이상, 입자 1-5μm 목표 |
| **계면활성제 조합** | 주 유화제 + 보조 유화제 혼합 사용 |
| **전해질 조절** | 과량 시 유화 파괴 → 0.5% 이하 유지 |
| **상 비율 최적화** | 오일상 15-25%가 밀크 클렌저 적정 범위 |

### 안정성 테스트
- 가속 시험: 40°C/75%RH, 3개월
- 동결-해동 사이클: -10°C ↔ 40°C, 5회 반복
- 원심분리: 3,000rpm, 30분 → 층분리 확인

## 2. HLB 밸런스

### HLB(Hydrophile-Lipophile Balance) 기본
| HLB 범위 | 용도 |
|----------|------|
| 3-6 | W/O 유화제 |
| 8-18 | O/W 유화제 |
| 13-16 | 세정제 |

### Required HLB 계산
```
Required HLB = Σ (오일 함량 비율 × 오일 Required HLB)
```

### 주요 오일의 Required HLB
| 오일 | Required HLB |
|------|-------------|
| 미네랄 오일 | 10-12 |
| 스쿠알란 | 11 |
| 호호바 오일 | 6-7 |
| 올리브 오일 | 7 |
| 코코넛 오일 | 8 |

### 유화제 HLB 값
| 유화제 | HLB |
|--------|-----|
| Polysorbate 80 | 15 |
| Polysorbate 20 | 16.7 |
| Cetearyl Olivate | 10 |
| Sorbitan Oleate | 4.3 |
| Glyceryl Stearate | 3.8 |

### 혼합 HLB 계산
```
혼합 HLB = (A% × HLB_A + B% × HLB_B) / (A% + B%)
```

**예시**: Polysorbate 80 (HLB 15) 2% + Glyceryl Stearate (HLB 3.8) 3%
```
= (2 × 15 + 3 × 3.8) / 5 = 8.28
```

## 3. 미백 성분 배합 전략

### 주요 미백 성분 및 특성
| 성분 | 농도 | pH | 주의사항 |
|------|------|-----|---------|
| **Niacinamide** | 2-5% | 5-7 | 비타민C와 분리 권장 |
| **Arbutin (α)** | 1-2% | 5-7 | 가수분해 주의 |
| **Ascorbyl Glucoside** | 2% | 5-7 | 안정성 우수 |
| **Tranexamic Acid** | 2-3% | 4-6 | 수용성, 안정 |
| **Kojic Acid** | 1-2% | 4-5 | 변색 주의 |
| **Ethyl Ascorbyl Ether** | 1-3% | 4-7 | 유도체 중 가장 안정 |

### 배합 시 고려사항

**1) pH 조절**
- 대부분 미백 성분: pH 5-6 최적
- 클렌저 사용 후 바로 씻어내므로 자극 덜함

**2) 성분 조합**
| 조합 | 시너지 |
|------|--------|
| Niacinamide + Tranexamic Acid | ✅ 우수 |
| Arbutin + Niacinamide | ✅ 우수 |
| Vitamin C + Niacinamide | ⚠️ pH 차이, 별도 배합 권장 |

**3) 클렌저 특수성**
- 접촉 시간 짧음 (30초-1분) → 침투 효과 제한적
- **전략**: 마케팅 소구점 + 리브온 제품과 라인 구성

**4) 안정화**
- 항산화제 병용: Tocopherol, BHT (0.05-0.1%)
- 차광 용기 사용
- 질소 치환 충전

### 권장 기본 처방 예시
```
[수상]
정제수 to 100%
Glycerin 5%
Niacinamide 3%
Tranexamic Acid 2%

[유상]
Squalane 10%
Cetearyl Alcohol 2%
Glyceryl Stearate 2%
Polysorbate 60 1.5%

[기타]
Carbomer 0.2%
Phenoxyethanol 0.8%
Tocopherol 0.1%
```
