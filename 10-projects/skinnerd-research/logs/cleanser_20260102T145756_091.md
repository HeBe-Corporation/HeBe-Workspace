# 젤 클렌저 제형 가이드

## 1. 투명도 유지 방법

**핵심 원칙**: 굴절률 매칭 + 용해도 최적화

| 요소 | 방법 |
|------|------|
| **계면활성제 선택** | 비이온성(코카미도프로필베타인) 또는 음이온성(소듐라우릴설포아세테이트) 사용 |
| **겔화제 농도** | 과량 사용 시 탁해짐 → 최소 유효 농도 유지 |
| **pH 조절** | 대부분 5.0-6.5에서 투명도 최적 |
| **전해질 관리** | 염(NaCl) 과량 → 미셀 크기 증가 → 탁해짐 |
| **온도 관리** | 제조 시 60°C 이하 유지, 급냉 피하기 |
| **탈기** | 기포 제거 필수 (진공 탈기 또는 정치) |

## 2. 겔화제 종류별 특성

### 수용성 겔화제

| 겔화제 | 농도 | 특성 | 투명도 |
|--------|------|------|--------|
| **Carbomer (카보머)** | 0.2-0.5% | 고점도, pH 6-7.5 필요, 중화 필수 | ★★★★★ |
| **Hydroxyethylcellulose (HEC)** | 0.5-2% | 염에 안정, 저자극 | ★★★★☆ |
| **Sodium PCA** | 점도조절 보조 | 보습 효과 겸비 | ★★★★★ |
| **Acrylates/C10-30 Alkyl Acrylate Crosspolymer** | 0.1-0.3% | 계면활성제 안정화 | ★★★★★ |
| **Xanthan Gum (잔탄검)** | 0.1-0.5% | 가성비 좋음, 살짝 탁함 | ★★★☆☆ |

### 카보머 중화제 선택
- **TEA (트리에탄올아민)**: 가장 투명, 하지만 안정성 논란
- **Sodium Hydroxide**: 투명도 양호
- **AMP (아미노메틸프로판올)**: 투명도 우수, 안정적

## 3. 미백 성분 배합 시 주의사항

### 성분별 주의점

| 성분 | pH 범위 | 주의사항 |
|------|---------|----------|
| **나이아신아마이드** | 5.0-7.0 | 비교적 안정, 고농도(4%+)시 홍조 가능 |
| **알부틴** | 5.0-7.0 | 열/빛에 분해 → 저온 제조, 차광 용기 |
| **비타민C (AA)** | 2.5-3.5 | 산화 취약, 겔과 pH 충돌 가능 |
| **비타민C 유도체 (SAP, MAP)** | 6.0-7.0 | 안정적, 겔 클렌저에 적합 |
| **트라넥삼산** | 4.5-6.5 | 안정적, 겔과 호환 좋음 |
| **코직산** | 4.0-5.0 | 산화되면 갈변, 킬레이트제 필수 |

### 공통 주의사항

1. **산화 방지**: EDTA, 피틱산 등 킬레이트제 0.05-0.1% 첨가
2. **차광**: 갈색/불투명 용기 또는 UV 안정제 사용
3. **온도**: 미백 성분은 40°C 이하에서 후첨가
4. **계면활성제 상호작용**: 음이온 계면활성제 + 양이온 성분 = 침전 위험
5. **사용 기한**: 미백 성분 함유 시 개봉 후 사용기한 단축 고려

### 추천 조합 (안정성 기준)
```
나이아신아마이드 2-4% + 트라넥삼산 0.5-2%
→ pH 5.5-6.0 / 카보머 겔 베이스와 호환 우수
```
