# 젤 클렌저 제형 가이드

## 1. 투명도 유지 방법

### 핵심 원칙
- **굴절률 매칭**: 물(1.33)과 계면활성제, 겔화제의 굴절률을 최대한 맞춤
- **완전 용해**: 모든 성분이 완전히 녹아야 함 (미용해 입자 = 혼탁)
- **pH 관리**: 대부분 pH 5.5-7.0에서 최적 투명도

### 실무 팁
| 요인 | 투명도 저하 원인 | 해결책 |
|------|------------------|--------|
| 온도 | 급냉 시 결정화 | 서서히 냉각 (40→25°C) |
| 염 농도 | 과량 NaCl | 0.5-2% 범위 유지 |
| 오일 성분 | 유화 불량 | 가용화제(PEG-40 Hydrogenated Castor Oil) 사용 |
| 향료 | 불투명 향료 | 수용성/투명 향료 선택 |

---

## 2. 겔화제 종류별 특성

### 합성 겔화제
| 겔화제 | 사용량 | 특징 | 주의점 |
|--------|--------|------|--------|
| **Carbomer** (940, 980) | 0.2-0.5% | 투명도 우수, 점도 높음 | 중화제 필수 (TEA, NaOH) |
| **Acrylates/C10-30 Alkyl Acrylate Crosspolymer** | 0.2-0.4% | 전해질 내성 좋음 | Carbomer보다 비쌈 |
| **PVM/MA Copolymer** | 0.5-1.5% | 필름 형성 | 끈적임 있음 |

### 천연 겔화제
| 겔화제 | 사용량 | 특징 | 주의점 |
|--------|--------|------|--------|
| **Xanthan Gum** | 0.3-1.0% | 전해질 안정, 슈도플라스틱 | 약간 혼탁, 분산 어려움 |
| **Hydroxyethylcellulose (HEC)** | 0.5-2.0% | 투명도 양호, 비이온성 | 점도 낮음, 미생물 취약 |
| **Sodium Hyaluronate** | 0.1-0.5% | 보습 기능 추가 | 고가, 점도 낮음 |

### 선택 가이드
```
투명도 중시 → Carbomer 940 + HEC 병용
전해질 함유 → Acrylates Crosspolymer
천연 컨셉 → Xanthan + HEC 조합
```

## 3. 미백 성분 배합 시 주의사항

### 성분별 주의점

| 미백 성분 | pH 조건 | 주의사항 |
|-----------|---------|----------|
| **Niacinamide** | pH 5-7 | 열 안정, Carbomer와 호환 좋음 |
| **Arbutin (α, β)** | pH 5-7 | 가수분해 주의, α가 더 안정 |
| **Ascorbic Acid (비타민C)** | pH 2.5-3.5 | 산화 취약, 갈변 위험 |
| **Tranexamic Acid** | pH 4-6 | 비교적 안정 |
| **Kojic Acid** | pH 4-5 | 변색 (노란→갈색), 킬레이트제 필요 |

### 배합 금지/주의 조합

```
❌ 피해야 할 조합:
- Niacinamide + 순수 Ascorbic Acid (pH 차이, 플러싱 반응)
- Arbutin + 강산/강알칼리 (가수분해)
- Kojic Acid + 금속 이온 (변색)

✅ 권장 조합:
- Niacinamide + Tranexamic Acid (상호 보완)
- α-Arbutin + Niacinamide (안정적)
```

### 안정화 전략

1. **항산화제 추가**: Tocopherol, Ferulic Acid, BHT
2. **킬레이트제**: EDTA-2Na, Phytic Acid (0.05-0.1%)
3. **차광 용기**: 갈색/불투명 용기 사용
4. **질소 충전**: 산화 방지
5. **낮은 수분활성도**: 글리세린, 부틸렌글라이콜 10-15%

## 배합 예시 (투명 미백 젤 클렌저)

```
Phase A (수상)
- 정제수 to 100%
- Glycerin 5%
- Niacinamide 2%
- EDTA-2Na 0.05%

Phase B (겔화)
- Carbomer 940 0.3%
- HEC 0.2%

Phase C (계면활성제)
- Cocamidopropyl Betaine 8%
- Sodium Lauryl Sulfoacetate 4%

Phase D (중화/조정)
- Triethanolamine q.s. (pH 5.5-6.0)
- 방부제, 향료

→ 순서: A 용해 → B 분산 → C 혼합 → D 중화
```

추가로 특정 성분이나 제형에 대해 더 알고 싶으면 말해줘.
