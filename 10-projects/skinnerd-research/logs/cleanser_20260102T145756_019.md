## 응답

# 휘핑 클렌저(Whipped Cleanser) 상세 가이드

## 제조 원리

### 1. 에멀전 + 공기 혼입 시스템
휘핑 클렌저는 **O/W(수중유적형) 에멀전**에 공기를 물리적으로 혼입시켜 만든다.

```
기본 구조:
수상(Water Phase) + 유상(Oil Phase) + 계면활성제 + 공기(Air)
         ↓
    고속 교반/휘핑
         ↓
    폼 형태의 안정화된 에멀전
```

### 2. 핵심 메커니즘
- **기포 형성**: 고속 믹싱으로 미세 기포를 유상/수상 경계면에 포집
- **계면 안정화**: 계면활성제가 기포 표면을 코팅해 붕괴 방지
- **점도 조절**: 증점제가 기포 이동을 억제해 구조 유지

## 핵심 성분

### 계면활성제 (Surfactants)
| 성분 | 역할 | HLB |
|------|------|-----|
| Cocamidopropyl Betaine | 마일드한 세정, 기포 안정 | 양쪽성 |
| Sodium Cocoyl Isethionate | 크리미한 거품, 저자극 | 14-16 |
| Disodium Laureth Sulfosuccinate | 풍성한 거품 | 12-14 |
| Sodium Lauroyl Glutamate | 아미노산계, 피부친화 | 12-14 |

### 유화제/안정제
| 성분 | 역할 |
|------|------|
| Glyceryl Stearate | O/W 유화, 크리미 텍스처 |
| Cetearyl Alcohol | 점도 증가, 폼 안정 |
| Stearic Acid | 구조 형성, 밀도감 |

### 증점제/폼 안정제
| 성분 | 역할 |
|------|------|
| Xanthan Gum | 수상 점도 증가 |
| Hydroxyethylcellulose | 폼 탄력 부여 |
| Carbomer | 겔 구조 형성 |
| Acrylates Copolymer | 기포막 강화 |

### 보습/컨디셔닝
| 성분 | 역할 |
|------|------|
| Glycerin | 수분 공급 |
| Shea Butter | 유분 보충, 부드러운 느낌 |
| Squalane | 피부 장벽 보호 |

## 텍스처 유지 방법

### 1. 제형 설계 단계
```
폼 안정성 최적화 조건:
- 계면활성제 농도: 8-15%
- 유상 비율: 5-15%
- 증점제: 0.3-1.0%
- pH: 5.5-6.5 (약산성)
```

### 2. 제조 공정
| 단계 | 조건 | 목적 |
|------|------|------|
| 수상 가열 | 70-75°C | 증점제 완전 용해 |
| 유상 가열 | 70-75°C | 유화제 용해 |
| 유화 | 3000-5000 rpm | 균일한 에멀전 형성 |
| 냉각 | 40°C까지 서냉 | 결정화 방지 |
| 휘핑 | 1500-2500 rpm | 공기 혼입 |
| 탈포 | 진공 -0.5bar | 큰 기포 제거, 미세 기포 유지 |

### 3. 보존 안정성 확보
```
필수 테스트:
- 가속 안정성: 40°C/75%RH, 3개월
- 동결-해동: -10°C ↔ 40°C, 5사이클
- 원심분리: 3000rpm, 30분 → 층분리 확인
```

### 4. 품질 유지 팁
- **용기**: 에어리스 펌프 또는 자립형 튜브 권장 (공기 추가 유입 방지)
- **사용 시**: 물 유입 최소화, 뚜껑 확실히 닫기
- **보관**: 직사광선/고온 피하기 (15-25°C 최적)

## 주의사항

1. **과도한 계면활성제 금지**: 기포는 잘 나지만 자극 증가
2. **실리콘 오일 주의**: 기포 안정성 저해 가능
3. **방부제 선택**: Phenoxyethanol + Ethylhexylglycerin 조합 권장
4. **향료**: 저알레르기 향료 사용, 0.3% 이하
