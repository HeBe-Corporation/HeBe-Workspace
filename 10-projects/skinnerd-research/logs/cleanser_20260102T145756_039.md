## 응답

# Tranexamic Acid (트라넥사믹애시드) 클렌저 적용 가이드

## 장점

1. **미백 효과 기대** - 멜라닌 생성 억제 (플라스민 활성 차단)
2. **순한 미백 성분** - 하이드로퀴논 대비 자극 적음
3. **수용성** - 물 기반 제형에 쉽게 용해
4. **안정성 양호** - 비타민C 대비 산화에 강함
5. **마케팅 포인트** - "세안하면서 미백" 컨셉 차별화

## 단점

1. **접촉 시간 짧음** - 클렌저는 1-2분 내 헹굼 → 침투/효능 제한적
2. **실질 효과 의문** - 대부분 연구가 leave-on 또는 경구 기준
3. **원가 상승** - 효과 대비 가성비 낮을 수 있음
4. **규제 이슈** - 국가별 미백 클레임 허용 농도 상이

## 최적 농도

| 제형 | 권장 농도 | 비고 |
|------|-----------|------|
| 클렌저 (wash-off) | **2-3%** | 접촉시간 짧아 높게 설정 |
| 세럼/크림 (leave-on) | 1-2% | 일반적 유효 농도 |
| 전문가용 | 3-5% | 병원/에스테틱 |

> 클렌저는 헹굼 제품이라 2-3%로 높여도 자극 위험 낮음

## 제형 안정성 확보 방법

### 1. pH 관리
- **최적 pH: 4.5-6.0** (약산성)
- 알칼리 환경에서 분해 가속 → pH 버퍼 시스템 필수
- 추천 버퍼: 시트르산/시트레이트 버퍼

### 2. 산화 방지
```
- 질소 퍼징 (제조/충전 시)
- 항산화제 첨가: BHT, 토코페롤 (소량)
- 차광 용기 사용
```

### 3. 온도 관리
- **보관: 상온 25°C 이하**
- 고온 노출 시 변색 가능 (황변)
- 가속 안정성 테스트: 40°C/75% RH, 3개월

### 4. 금속 이온 제거
- 킬레이트제 첨가: EDTA-2Na (0.05-0.1%)
- 금속 이온이 분해 촉진

### 5. 배합 주의
| 호환 ✓ | 비호환 ✗ |
|--------|----------|
| 나이아신아마이드 | 강산 (글리콜산 고농도) |
| 히알루론산 | 강알칼리 |
| 판테놀 | 산화제 |
| 알란토인 | 일부 금속염 |

## 실무 제언

1. **클렌저보다 토너/세럼 권장** - 실효성 면에서 leave-on이 우월
2. **클렌저 적용 시** - "미백 부스팅" 컨셉으로 라인 구성 (클렌저→토너→세럼 동일 성분)
3. **안정성 시험 필수** - 최소 6개월 장기, 3개월 가속
4. **클레임 검토** - 한국 식약처 미백 기능성 등록 여부 확인 필요
