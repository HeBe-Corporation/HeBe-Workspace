## 응답

# 클렌저 pH 최적화: 미백 성분별 전략

## 피부 산성막 기준

**건강한 피부 pH**: 4.5~5.5 (약산성)

클렌저는 세정 후 피부가 이 범위로 빠르게 회복할 수 있어야 함.

---

## 미백 성분별 최적 pH 범위

| 성분 | 최적 pH | 이유 |
|------|---------|------|
| **비타민 C (L-Ascorbic Acid)** | 2.5~3.5 | pH 3.5 이상에서 급격히 불안정화, 침투력 저하 |
| **나이아신아마이드 (B3)** | 5.0~7.0 | 산성에서 나이아신으로 전환 → 홍조 유발 |
| **알부틴** | 5.0~7.0 | 강산성에서 하이드로퀴논으로 분해 |
| **코직산** | 4.0~5.5 | 알칼리에서 산화/변색 |
| **트라넥사믹산** | 4.5~6.5 | 넓은 pH 안정성 |
| **아젤라익산** | 4.0~5.0 | 약산성에서 최적 침투 |
| **글루타치온** | 5.0~6.5 | 산성에서 산화 가속 |

---

## 밸런스 전략

### 1. 클렌저 타입별 권장 pH

| 타입 | 권장 pH | 비고 |
|------|---------|------|
| **폼 클렌저** | 5.5~6.5 | 계면활성제 기포력 유지 필요 |
| **젤 클렌저** | 4.5~5.5 | 피부 pH와 동일 유지 가능 |
| **밀크/크림 클렌저** | 5.0~6.0 | 유화 안정성 고려 |
| **오일 클렌저** | pH 무관 | 수상 없음 |

### 2. 성분 조합 시 충돌 회피

```
❌ 피해야 할 조합 (pH 충돌)
- 비타민C(pH 3) + 나이아신아마이드(pH 6) → 동시 사용 시 효능 저하

✅ 안전한 조합
- 트라넥사믹산 + 나이아신아마이드 (pH 5~6 공유)
- 코직산 + 아젤라익산 (pH 4~5 공유)
- 알부틴 + 글루타치온 (pH 5~6.5 공유)
```

### 3. 제형 전략

| 전략 | 설명 |
|------|------|
| **이중 세안** | 1차 오일(pH 무관) → 2차 젤/폼(pH 5.5) |
| **버퍼링** | 시트르산/젖산으로 pH 완충 |
| **성분 분리** | 미백 성분은 클렌저보다 **세럼/에센스**에 집중 |
| **pH 조절제** | 트리에탄올아민(상승), 시트르산(하강) |

---

## 실무 권장사항

### 클렌저 제형 시
1. **타겟 pH**: 5.0~5.5 (피부 친화적 + 대부분 미백성분 안정)
2. **비타민C**는 클렌저에 넣지 말 것 → 세럼으로 분리
3. **나이아신아마이드/트라넥사믹산**은 클렌저 적합

### 제품 개발 체크리스트
```
□ pH 미터로 실측 (리트머스 불충분)
□ 3개월 가속 안정성 테스트 (45°C)
□ 미백 효능 vs 세정력 트레이드오프 검토
□ 피부 pH 회복 시간 측정 (15분 내 정상화)
```

## 요약

> **클렌저 최적 pH: 5.0~5.5**

- 비타민C는 세럼으로 분리
- 나이아신아마이드/트라넥사믹산/알부틴은 클렌저 호환
- 세정 후 피부 pH 회복 속도가 핵심 지표
