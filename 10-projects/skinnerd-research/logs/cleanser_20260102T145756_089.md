## 응답

# 휘핑 클렌저 (Whipped Cleanser) 상세 가이드

## 1. 제조 원리

### 기본 메커니즘
휘핑 클렌저는 **공기를 유화 시스템에 포집**하여 가볍고 푹신한 질감을 만드는 제형이야.

```
오일상 + 수상 + 계면활성제 → 유화 → 고속 교반(휘핑) → 공기 포집 → 크림 같은 무스 질감
```

### 핵심 공정
1. **유화 (Emulsification)**: 오일과 물을 계면활성제로 안정화
2. **에어레이션 (Aeration)**: 고속 믹싱으로 미세 기포 주입
3. **안정화 (Stabilization)**: 기포가 붕괴되지 않도록 점증제/왁스로 구조 고정

---

## 2. 핵심 성분

### 계면활성제 (Surfactants)
| 성분 | 역할 | 특징 |
|------|------|------|
| Sodium Cocoyl Isethionate | 주 세정제 | 저자극, 크리미한 거품 |
| Cocamidopropyl Betaine | 보조 세정제 | 거품 안정화, 자극 완화 |
| Sodium Lauroyl Sarcosinate | 아미노산계 | 순한 세정, pH 안정 |

### 유화제/안정제
| 성분 | 역할 |
|------|------|
| Glyceryl Stearate | O/W 유화제, 크리미한 질감 |
| Cetearyl Alcohol | 점도 조절, 유화 안정 |
| Stearic Acid | 질감 강화, 거품 밀도 증가 |

### 기포 안정제
| 성분 | 역할 |
|------|------|
| Xanthan Gum | 기포 붕괴 방지, 점도 유지 |
| Hydroxyethylcellulose | 구조 안정화 |
| Beeswax / Carnauba Wax | 왁스 매트릭스로 공기 포집 |

### 보습제
- **Glycerin**: 수분 공급, 세정 후 당김 방지
- **Squalane**: 피부 장벽 보호
- **Hyaluronic Acid**: 보습막 형성

---

## 3. 텍스처 유지 방법

### 제조 시 핵심 포인트

```
① 온도 관리
   - 유화 온도: 70-75°C (오일상/수상 동일)
   - 휘핑 온도: 40-45°C (냉각 후 휘핑)
   - 너무 뜨거우면 기포 유지 불가

② 휘핑 조건
   - 고속 믹서 사용 (플래니터리 믹서 권장)
   - 3,000-5,000 RPM에서 5-10분
   - 과도한 휘핑은 기포 붕괴 유발

③ 점증제 투입 타이밍
   - 휘핑 전에 점증제 완전 수화
   - 수화 불량 시 기포 안정성 저하
```

### 보관/유통 안정성
| 요인 | 대응 방법 |
|------|-----------|
| 온도 변화 | 왁스 함량 증가 (1-3%) |
| 시간 경과 | Xanthan + Cellulose 복합 사용 |
| 용기 문제 | 에어리스 펌프 또는 자펫 용기 |
| 미생물 | 적절한 방부 시스템 (pH 5.5-6.5 유지) |

### 실패 원인과 해결

| 문제 | 원인 | 해결 |
|------|------|------|
| 기포 붕괴 | 점증제 부족, 고온 휘핑 | 점증제 0.3-0.5% 증량, 냉각 후 휘핑 |
| 분리 현상 | 유화 불안정 | 유화제 비율 조정, HLB 값 확인 |
| 너무 무거움 | 왁스 과다 | 왁스 감량, 경질 오일 대체 |
| 건조함 | 계면활성제 과다 | 보습제 증량, 순한 계면활성제 교체 |
