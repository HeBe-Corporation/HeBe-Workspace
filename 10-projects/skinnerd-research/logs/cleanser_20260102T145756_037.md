# 클렌저 리서치 #37
- 시간: 2026. 1. 3. 오후 6:24:43

## 질문
Niacinamide를 클렌저에 적용할 때의 최적 농도(3-5% vs 10%), 다른 성분과의 호환성, 세정 후 피부 잔류율을 분석해줘.

## 응답

웹 검색 권한이 없어서 기존 지식으로 분석할게.

---

## 1. 최적 농도: 3-5% vs 10%

### 클렌저에서의 권장: **3-5%**

| 항목 | 3-5% | 10% |
|------|------|-----|
| **클렌저 적합성** | ✅ 최적 | ⚠️ 과잉 |
| **비용 효율** | 높음 | 낮음 (대부분 씻겨나감) |
| **피부 자극** | 거의 없음 | 일부 민감성 피부 자극 가능 |
| **효능 발현** | 충분 | 잔류 시간 부족으로 10% 의미 없음 |

### 이유
- **접촉 시간**: 클렌저는 30초~2분 사용 후 세정 → 고농도 의미 감소
- **연구 기반**: Leave-on 제품에서 2-5%로도 피지 조절, 모공 개선 효과 입증
- **경제성**: 10%는 원가 상승 대비 효과 증가 미미

---

## 2. 성분 호환성

### ✅ 호환 가능
| 성분 | 시너지 효과 |
|------|------------|
| **Hyaluronic Acid** | 보습 강화 |
| **Zinc PCA** | 피지 조절 상승 |
| **Panthenol** | 진정 + 장벽 강화 |
| **Glycerin** | 수분 유지 |
| **Salicylic Acid (BHA)** | pH 4-6 범위에서 안정 |

### ⚠️ 주의 필요
| 성분 | 문제점 | 해결책 |
|------|--------|--------|
| **Vitamin C (L-AA)** | 낮은 pH(2.5-3.5)에서 Niacinamide → Niacin 전환 (홍조 유발) | 별도 제품으로 분리 또는 유도체 사용 |
| **AHA (Glycolic 등)** | pH 3-4에서 동일 문제 | 클렌저 pH 5-6 유지 시 OK |
| **Copper Peptides** | 상호작용 논쟁 있음 | 분리 사용 권장 |

### 최적 pH
- Niacinamide 안정성: **pH 5.0-7.0**
- 클렌저 권장 pH: **5.5-6.5** (피부 장벽 친화적)

---

## 3. 세정 후 피부 잔류율

### 예상 잔류율: **5-15%**

| 요인 | 영향 |
|------|------|
| **접촉 시간** | 길수록 흡수↑ (마사지 1-2분 권장) |
| **제형** | 폼 < 크림 < 오일 클렌저 (오일 타입이 잔류율 높음) |
| **물 온도** | 미온수가 최적 (뜨거운 물은 씻김↑) |
| **피부 상태** | 손상된 피부 > 건강한 피부 (흡수율) |

### 잔류 효과 극대화 전략
1. **"60초 룰"**: 최소 60초 마사지 후 헹굼
2. **이중 세안 시**: 2차 클렌저에 Niacinamide 배합
3. **부스터 역할**: Leave-on 제품(토너/세럼)과 병행 시 효과 극대화

## 결론 및 제안

### 클렌저 포뮬레이션 권장
```
Niacinamide: 3-5%
pH: 5.5-6.0
병용 성분: Zinc PCA, Panthenol, HA
피해야 할 성분: L-Ascorbic Acid, 고농도 AHA
```

### 마케팅 포인트
- "세정하면서 피부결 케어"
- "피지 밸런스 클렌저"
- 단, "10% 나이아신아마이드!" 강조는 과대광고 소지 → 3-5%로도 충분히 효과적

---

**핵심**: 클렌저는 wash-off 제품이므로 고농도보다 **적정 농도(3-5%) + 충분한 마사지 시간**이 더 중요해. 10%는 세럼/크림 같은 leave-on 제품에 적합.
