## 응답

# 버블 클렌저 vs 폼 클렌저 비교 분석

## 핵심 차이점

| 구분 | 버블 클렌저 | 폼 클렌저 |
|------|------------|----------|
| **거품 생성** | 펌프에서 즉시 거품 배출 | 손으로 직접 거품을 만들어야 함 |
| **주요 기술** | 에어 펌프 + 계면활성제 희석액 | 고농축 계면활성제 페이스트/젤 |
| **pH 범위** | 대체로 5.5~6.5 (약산성) | 제품별 편차 큼 (5.0~9.0) |

---

## 1. 거품 생성 메커니즘

### 버블 클렌저
- **에어믹싱 펌프**: 공기와 액체를 1:10~1:20 비율로 혼합
- **계면활성제 농도**: 5~10%로 낮음 (이미 희석된 상태)
- **거품 특성**: 균일하고 미세한 기포, 밀도 낮음

### 폼 클렌저
- **물리적 기포 형성**: 손바닥 마찰 + 물 + 공기
- **계면활성제 농도**: 15~30%로 높음
- **거품 특성**: 사용자 기술에 따라 다름, 밀도 높은 거품 가능

---

## 2. 세정력 비교

| 항목 | 버블 클렌저 | 폼 클렌저 |
|------|------------|----------|
| 피지 제거 | ★★★☆☆ | ★★★★☆ |
| 메이크업 잔여물 | ★★☆☆☆ | ★★★★☆ |
| 미세먼지/노폐물 | ★★★☆☆ | ★★★★☆ |
| 모공 딥클렌징 | ★★☆☆☆ | ★★★★☆ |

**결론**: 폼 클렌저가 세정력 우위 (고농축 계면활성제)

---

## 3. 피부 자극도

### 버블 클렌저
- **장점**: 낮은 계면활성제 농도 → 피부 장벽 손상 최소화
- **pH**: 대부분 약산성으로 제조
- **자극도**: ★★☆☆☆ (낮음)

### 폼 클렌저
- **위험요소**: 고농축 성분, 알칼리성 제품 다수
- **SLS/SLES**: 일부 제품에 포함 (자극 유발 가능)
- **자극도**: ★★★☆☆~★★★★☆ (제품별 편차 큼)

---

## 4. 장단점 정리

### 버블 클렌저
| 장점 | 단점 |
|------|------|
| 즉시 사용 가능 (편의성) | 세정력 약함 |
| 피부 자극 적음 | 진한 메이크업 제거 어려움 |
| 민감성/건성 피부에 적합 | 용량 대비 가격 높음 |
| 거품 품질 일정 | 펌프 고장 시 사용 불가 |

### 폼 클렌저
| 장점 | 단점 |
|------|------|
| 강력한 세정력 | 과세정 위험 |
| 지성/복합성 피부에 효과적 | 제품 선택 중요 (pH 확인 필수) |
| 용량 대비 경제적 | 거품 만드는 시간 필요 |
| 딥클렌징에 적합 | 민감성 피부 주의 |

---

## 5. 피부 타입별 추천

| 피부 타입 | 추천 | 이유 |
|----------|------|------|
| 건성/민감성 | **버블 클렌저** | 낮은 자극, 수분 보존 |
| 지성/복합성 | **폼 클렌저** | 피지 제거력 필요 |
| 여드름성 | 폼 클렌저 (약산성) | 모공 딥클렌징 |
| 노메이크업 | 버블 클렌저 | 가벼운 세정으로 충분 |
| 풀메이크업 | 더블클렌징 필수 | 오일 → 폼/버블 |

---

## 핵심 요약

```
세정력: 폼 > 버블
자극도: 폼 > 버블 (폼이 더 자극적)
편의성: 버블 > 폼
경제성: 폼 > 버블
```

**선택 기준**: 피부 타입과 메이크업 농도에 따라 결정. 민감성이면 버블, 지성이면 폼.
