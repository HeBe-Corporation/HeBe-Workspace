## 응답

# Niacinamide 클렌저 적용 분석

## 최적 농도: 3-5% vs 10%

### 3-5% 권장 (클렌저용)

| 항목 | 3-5% | 10% |
|------|------|-----|
| **세정 후 잔류** | 충분한 효과 | 대부분 씻겨나감 (과잉) |
| **자극 가능성** | 낮음 | 피부 홍조/자극 위험 |
| **비용 효율** | 높음 | 낭비 |
| **pH 안정성** | 우수 | 제형 불안정 가능 |

**결론**: 클렌저는 피부 접촉 시간이 30초-1분으로 짧아 **3-5%가 최적**. 10%는 Leave-on 제품(세럼, 크림)에 적합.

---

## 성분 호환성

### ✅ 호환 가능
| 성분 | 시너지 효과 |
|------|-------------|
| **히알루론산** | 보습 강화 |
| **글리세린** | 수분 유지 |
| **판테놀** | 진정 + 장벽 강화 |
| **세라마이드** | 장벽 회복 |
| **아미노산 계면활성제** | 저자극 세정 |

### ⚠️ 주의 필요
| 성분 | 문제점 | 해결책 |
|------|--------|--------|
| **비타민 C (L-AA)** | pH 차이로 효능 저하 | 별도 사용 권장 |
| **AHA/BHA** | 산성 pH에서 나이아신 전환 위험 | pH 5.5 이상 유지 |
| **구리펩타이드** | 상호작용으로 효능 감소 | 함께 사용 피함 |

### ❌ 비호환
| 성분 | 이유 |
|------|------|
| **순수 레티놀** | 제형 불안정, 자극 증가 |
| **벤조일퍼옥사이드** | 산화로 효능 감소 |

## 세정 후 피부 잔류율

### 연구 기반 데이터
```
접촉 시간 30초 → 잔류율 약 5-10%
접촉 시간 60초 → 잔류율 약 10-15%
마사지 강도 ↑   → 잔류율 소폭 증가
```

### 실질적 효과
| 효과 | 클렌저 적용 시 기대치 |
|------|----------------------|
| 피지 조절 | △ 제한적 |
| 모공 관리 | △ 보조적 |
| 미백 | ✗ 거의 없음 |
| 장벽 강화 | △ 미미 |
| 항염 | ○ 일부 효과 |

---

## 제형 권장사항

```
pH: 5.0 - 7.0 (최적 5.5-6.5)
Niacinamide: 3-5%
계면활성제: 아미노산 기반 (저자극)
추가 권장: 글리세린 2-3%, 판테놀 0.5-1%
```

### 핵심 포인트
1. **클렌저에서 10%는 과잉** - 씻겨나가므로 비용 낭비
2. **진정한 Niacinamide 효과**는 세럼/에센스에서 기대
3. **클렌저 역할**: 세정 중 자극 완화 + 후속 제품 흡수 준비

---

추가로 특정 계면활성제와의 조합이나 제형 안정성 테스트 방법이 필요하면 알려줘.
