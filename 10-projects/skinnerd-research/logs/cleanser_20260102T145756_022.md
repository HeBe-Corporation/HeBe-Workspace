## 응답

# 오일 클렌저에 수용성 미백 성분 배합 기술

오일 클렌저는 연속상이 오일이라 수용성 성분(나이아신아마이드, 비타민C 유도체, 알부틴 등)을 안정적으로 배합하기 어려움. 주요 해결 기술들:

---

## 1. 역상 에멀전 (W/O Emulsion)

**원리**: 오일 연속상 안에 수용성 성분을 담은 미세 물방울 분산

```
[오일 연속상] ← 계면활성제 ← [수용성 성분 + 물]
                              (내부상)
```

**핵심 요소**:
- HLB 3~6의 저HLB 유화제 사용 (Sorbitan Oleate, Polyglycerol-3 Polyricinoleate)
- 물방울 크기 1~10μm 유지
- 점도 조절로 크리밍/합일 방지

---

## 2. 나노에멀전 (Nanoemulsion)

**원리**: 물방울 크기를 50~200nm로 극소화 → 열역학적 안정성 확보

**제조법**:
| 방법 | 특징 |
|------|------|
| 고압 균질화 | 500~2000 bar 압력, 산업 표준 |
| 초음파 처리 | 실험실 규모, 열 발생 주의 |
| 저에너지법 (PIT/PIC) | 상전이 활용, 열에 민감한 성분에 적합 |

**장점**: 투명/반투명 외관, 피부 침투력 향상, 장기 안정성

---

## 3. 마이크로캡슐화 (Microencapsulation)

**원리**: 수용성 성분을 고분자 벽으로 감싸서 오일상과 물리적 격리

**주요 벽재**:
```
- 젤라틴/아라비아검 (코아세르베이션)
- 말토덱스트린 (분무건조)
- 사이클로덱스트린 (포접복합체)
- PLGA/PCL (생분해성 폴리머)
```

**방출 메커니즘**: 물과 접촉 시 벽 용해 → 세안 중 성분 방출

---

## 4. 리포좀/나노좀 (Liposome)

**원리**: 인지질 이중막이 수용성 코어를 감싸는 구조

```
      ┌─────────────────┐
      │ 인지질 이중막   │
      │   ┌─────────┐   │
      │   │수용성성분│   │
      │   │ (코어)  │   │
      │   └─────────┘   │
      └─────────────────┘
```

**특징**: 피부 친화성 우수, 리포좀 자체가 오일상에 분산 가능

---

## 5. 사이클로덱스트린 포접 (Inclusion Complex)

**원리**: 도넛 모양 분자의 소수성 공동에 성분 삽입

- α, β, γ-사이클로덱스트린 중 성분 크기에 맞는 것 선택
- 수용성 성분도 소수성 부위가 있으면 포접 가능
- 분말 형태로 오일에 분산

---

## 6. 실리카 흡착 (Silica Adsorption)

**원리**: 다공성 실리카에 수용액 흡착 → 분말화 → 오일 분산

```
수용성 성분 용액 → 다공성 실리카 흡착 → 건조 → 오일에 분산
```

**장점**: 간단한 공정, 비용 효율적

---

## 기술 선택 가이드

| 기술 | 투명도 | 안정성 | 비용 | 적합 성분 |
|------|--------|--------|------|-----------|
| W/O 에멀전 | 불투명 | 중 | 저 | 대부분 |
| 나노에멀전 | 투명 | 고 | 중 | 열안정 성분 |
| 마이크로캡슐 | 불투명 | 고 | 고 | 불안정 성분 |
| 리포좀 | 반투명 | 중 | 고 | 고급 라인 |
| CD 포접 | 투명 | 고 | 중 | 소분자 |

---

## 실무 고려사항

1. **pH 관리**: 캡슐/에멀전 내부 pH 버퍼링 필요
2. **산화 방지**: 비타민C 유도체는 불활성 가스 충전 권장
3. **출시 안정성**: 40°C 8주 가속 시험 필수
4. **사용감**: 캡슐 크기가 크면 이질감 발생 가능
