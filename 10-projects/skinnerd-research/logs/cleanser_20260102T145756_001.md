## 응답

# Glutathione 미백 클렌저 분석

## 1. 최적 농도

| 용도 | 농도 범위 | 비고 |
|------|-----------|------|
| 클렌저 (wash-off) | **0.1-1.0%** | 접촉시간 짧아 고농도 필요 |
| 리브온 제품 | 0.05-0.5% | 장시간 접촉으로 저농도 가능 |
| 임상 효과 확인 | **0.5% 이상** | 대부분 연구에서 유의미한 결과 |

클렌저는 **wash-off 제품**이라 피부 접촉 시간이 30초-2분으로 짧음. 효과를 보려면 **0.5-1.0%** 권장.

---

## 2. 안정화 방법

Glutathione(GSH)은 **산화에 매우 취약**함. 주요 안정화 전략:

### 2.1 유도체 사용 (권장)
| 유도체 | 장점 | 농도 |
|--------|------|------|
| **Glutathione Liposome** | 피부 침투↑, 안정성↑ | 0.5-2% |
| **Oxidized Glutathione (GSSG)** | 안정적, 피부에서 GSH로 환원 | 0.1-0.5% |
| **Acetyl Glutathione** | 지용성↑, 침투↑ | 0.1-0.5% |

### 2.2 제형적 안정화
```
- 항산화제 병용: Vitamin C (Ascorbyl Glucoside), Vitamin E, Ferulic Acid
- 금속이온 킬레이트: EDTA, Phytic Acid 첨가
- 질소 충전: 제조/충전 시 질소 퍼지
- 차광 용기: 갈색/불투명 용기 사용
- 저온 보관: 25°C 이하 권장
```

### 2.3 캡슐화 기술
- Liposome encapsulation
- Cyclodextrin inclusion complex
- 실리카 마이크로캡슐

## 3. pH 조건

| 조건 | pH | 이유 |
|------|-----|------|
| **GSH 안정 범위** | **3.0-5.5** | pH 6 이상에서 급격히 산화 |
| 클렌저 최적 pH | **4.5-5.5** | 피부 자극 최소화 + 안정성 |
| 피부 장벽 유지 | 4.5-5.5 | 피부 천연 pH와 유사 |

**주의**: pH 3 미만은 피부 자극, pH 6 이상은 GSH 분해 가속화

---

## 4. 클렌저에서의 효능 발현 메커니즘

### 4.1 미백 메커니즘
```
Glutathione의 이중 경로:

경로 1: Tyrosinase 억제
  GSH → Tyrosinase 구리이온(Cu²⁺)에 결합 → 효소 활성 억제 → 멜라닌 생성↓

경로 2: Pheomelanin 전환
  GSH + Dopaquinone → Cysteinyldopa → Pheomelanin (황색/적색)
  (Eumelanin 흑갈색 → Pheomelanin 황적색 전환)
```

### 4.2 클렌저 특유의 한계와 대응

| 한계 | 대응 전략 |
|------|-----------|
| 짧은 접촉시간 | 고농도(0.5-1%) + 침투촉진제(PEG-40 Hydrogenated Castor Oil) |
| 계면활성제와 상호작용 | 비이온 계면활성제 우선, 음이온 계면활성제 최소화 |
| Wash-off로 잔류량↓ | 리브온 제품과 연계 사용 권장 |

### 4.3 시너지 성분 조합
```
추천 조합:
- GSH 0.5-1% + Niacinamide 2-4% → 멜라닌 전달 억제 시너지
- GSH + Alpha Arbutin 1-2% → Tyrosinase 이중 억제
- GSH + Vitamin C 유도체 → 항산화 + GSH 재생 촉진
```

## 5. 처방 예시 (클렌저)

```
Phase A (수상)
  정제수 ..................... to 100%
  Glycerin ................... 5%
  Disodium EDTA .............. 0.1%
  
Phase B (유효성분)
  Glutathione (또는 Liposome형) . 0.5-1%
  Niacinamide ................ 2%
  
Phase C (계면활성제)
  Cocamidopropyl Betaine ..... 8%
  Sodium Cocoyl Isethionate .. 5%
  PEG-40 Hydrogenated Castor Oil 2%
  
Phase D (안정화)
  Phenoxyethanol ............. 0.8%
  Sodium Benzoate ............ 0.2%
  Citric Acid ................ pH 4.5-5.0으로 조정
```

## 핵심 요약

| 항목 | 권장값 |
|------|--------|
| **농도** | 0.5-1.0% |
| **pH** | 4.5-5.5 |
| **안정화** | Liposome형 유도체 + EDTA + 차광용기 |
| **시너지** | Niacinamide, Alpha Arbutin, Vitamin C |

클렌저는 wash-off 특성상 **단독 미백 효과 기대 어려움** → 세럼/크림 등 리브온 제품과 함께 사용하는 레지멘 설계 필요.
