## 동남아 미백 화장품 규제 요약

### 🇻🇳 베트남

**규제 기관**: 보건부 (Ministry of Health) - 의약품청

**금지 성분**:
| 성분 | 비고 |
|------|------|
| 수은(Mercury) 및 수은 화합물 | 전면 금지 |
| 하이드로퀴논(Hydroquinone) | 2% 초과 금지 (처방전 필요) |
| 트레티노인(Tretinoin) | 처방 의약품으로 분류 |
| 코르티코스테로이드(Corticosteroids) | 화장품 사용 금지 |

**허용 성분 및 농도**:
| 성분 | 최대 허용 농도 |
|------|----------------|
| 알부틴(Arbutin) | 7% |
| 코직산(Kojic Acid) | 1% |
| 아스코르빈산(Vitamin C) | 제한 없음 (안정성 기준 충족 시) |
| 나이아신아마이드(Niacinamide) | 제한 없음 |

**필수 표기사항**:
- 베트남어 라벨 필수
- 전성분 표기 (INCI 기준)
- 제조일/유통기한
- 수입업체/제조사 정보
- 공고번호(Notification Number)

### 🇹🇭 태국

**규제 기관**: 태국 FDA (Food and Drug Administration)

**금지 성분**:
| 성분 | 비고 |
|------|------|
| 수은(Mercury) | 전면 금지 |
| 하이드로퀴논(Hydroquinone) | 2% 초과 금지 |
| 스테로이드(Steroids) | 전면 금지 |
| 레티노이드(Retinoids) - 레티놀 제외 | 처방 필요 |

**허용 성분 및 농도**:
| 성분 | 최대 허용 농도 |
|------|----------------|
| 알부틴(Arbutin) | 7% |
| 코직산(Kojic Acid) | 1% |
| 글루타치온(Glutathione) | 제한 없음 (외용) |
| AHA/BHA | 10% / 2% (pH 3.5 이상) |

**필수 표기사항**:
- 태국어 라벨 필수
- FDA 등록번호
- 전성분 (태국어 또는 영어)
- 제조국, 수입업체
- 사용기한/로트번호
- "Whitening" 효능 표시 시 증빙 필요

## ASEAN 공통 규정 (ASEAN Cosmetic Directive)

모든 ASEAN 회원국은 **ASEAN Cosmetic Directive(ACD)**를 기반으로 규제:

| 항목 | 내용 |
|------|------|
| 금지 성분 | Annex II (1,500+ 성분) |
| 제한 성분 | Annex III (농도/용도 제한) |
| 착색제 | Annex IV |
| 방부제 | Annex VI |
| 자외선차단제 | Annex VII |

## 핵심 주의사항

1. **수은(Mercury)**: 전 동남아 국가에서 전면 금지 - 위반 시 제품 압수 및 형사처벌
2. **하이드로퀴논**: 대부분 2% 이하만 허용, 인도네시아는 전면 금지
3. **미백 효능 광고**: 증빙 데이터 필수 (임상시험 or in-vitro 테스트)
4. **현지어 라벨**: 대부분 국가에서 필수
