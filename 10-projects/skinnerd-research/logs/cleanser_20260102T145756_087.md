## 동남아 'Glow' 클렌저 트렌드 분석

### 1. 핵심 성분 구성

| 카테고리 | 주요 성분 | 역할 |
|----------|----------|------|
| **각질 케어** | AHA (글리콜릭/락틱산), PHA, 파파인 | 부드러운 각질 제거, 피부결 개선 |
| **보습/광채** | 히알루론산, 글리세린, 스쿠알란 | 세안 후 당김 방지, 수분광 |
| **브라이트닝** | 나이아신아마이드, 비타민C 유도체, 알부틴 | 톤업, 잡티 케어 |
| **천연 추출물** | 쌀뜨물, 강황(터메릭), 파파야, 코코넛 | 로컬 성분 선호, 전통 미용법 연계 |
| **진정** | 센텔라, 알로에, 그린티 | 열대기후로 인한 민감성 케어 |

### 2. 동남아 특화 트렌드

**기후 맞춤형**
- 고온다습 환경 → 가볍고 산뜻한 젤/폼 타입 선호
- 오일 컨트롤 + 광채 동시 추구
- "Dewy but not oily" 밸런스 중시

**로컬 성분 인기**
- 🇻🇳 베트남: 쌀뜨물(Rice water), 녹두
- 🇹🇭 태국: 강황, 타마린드
- 🇵🇭 필리핀: 파파야, 칼라만시
- 🇮🇩 인도네시아: 템페, 자무(Jamu) 허브

### 3. 마케팅 키워드

**영어 키워드**
```
Glass Skin / Glow / Dewy / Radiant / Luminous
Brightening / Tone-up / Clear Skin
K-beauty / J-beauty
Gentle / Low pH / Non-stripping
```

**로컬라이즈 키워드**
| 국가 | 인기 키워드 |
|------|------------|
| 베트남 | Da căng bóng (탱탱 윤광), Trắng sáng (화이트닝) |
| 태국 | ผิวกระจ่างใส (맑고 투명한 피부), ผิวฉ่ำวาว (촉촉 광채) |
| 필리핀 | Glowing skin, Fresh look, Poreless |

**SNS 해시태그 트렌드**


### 4. 제품 포지셔닝 전략

| 전략 | 설명 |
|------|------|
| **즉각적 효과 강조** | "1회 사용 후 바로 광채" 클레임 |
| **멀티 기능** | 클렌징 + 톤업 + 보습 3-in-1 |
| **저자극 pH** | pH 5.5~6.0 약산성 강조 |
| **비건/클린뷰티** | 동물실험 프리, 자연유래 성분 |
| **가성비** | 용량 대비 가격, 데일리 사용 강조 |

### 5. 경쟁사 벤치마킹 포인트

**K-뷰티 인기 제품 공통점**
- 투명/반투명 젤 텍스처
- 저자극 계면활성제 (코코베타인 등)
- 세안 후 "쫀쫀한 수분막" 느낌
- 패키징: 깔끔한 화이트/투명 + 미니멀 디자인
