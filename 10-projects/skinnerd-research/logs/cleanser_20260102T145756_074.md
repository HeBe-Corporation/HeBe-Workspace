## 응답

# Tranexamic Acid 클렌저 적용 가이드

## 장단점

### 장점
- **짧은 접촉 시간에도 효과**: TXA는 수용성이 높아 짧은 시간 내 피부 침투 가능
- **자극 최소화**: 린스오프 제형이라 민감성 피부에도 적용 가능
- **일상 루틴 편입 용이**: 별도 단계 추가 없이 미백 케어 가능
- **안정성 우수**: TXA는 pH, 열, 광에 비교적 안정적

### 단점
- **효능 한계**: 접촉 시간이 30초~1분으로 짧아 리브온 제품 대비 효과 감소
- **용해도 제한**: 고농도 배합 시 석출 가능성
- **소비자 기대 관리**: "클렌저로 미백?"에 대한 마케팅 설득 필요

---

## 최적 농도

| 농도 | 적용성 | 비고 |
|------|--------|------|
| **2~3%** | 권장 | 효능-안정성 균형점 |
| 1% 미만 | 마케팅용 | 실효성 의문 |
| 5% | 가능하나 주의 | 용해도 한계 근접, pH 조절 필수 |

**참고**: 
- TXA 수용해도: 약 15% (25°C)
- 클렌저 수상 비율 고려 시 3~5%가 현실적 상한

---

## 제형 안정성 확보 방법

### 1. pH 관리
```
최적 pH: 4.5~6.0
- TXA pKa ≈ 4.3 (카르복실기), 10.6 (아미노기)
- 중성~약산성에서 안정
- pH 7 초과 시 장기 안정성 저하 가능
```

### 2. 용해도 확보
- **수상 비율 충분히 확보** (70% 이상 권장)
- 용해 보조제: Propylene Glycol, Butylene Glycol (5~10%)
- 가온 용해 후 서냉

### 3. 산화/변색 방지
- TXA 자체는 산화에 강하나, 공존 성분 고려
- 킬레이트제 추가: Disodium EDTA (0.05~0.1%)
- 질소 치환 충전 권장

### 4. 계면활성제 호환성
| 계면활성제 타입 | 호환성 | 비고 |
|----------------|--------|------|
| 음이온 (SLES, SLS) | 양호 | 일반적 조합 |
| 양쪽성 (Cocamidopropyl Betaine) | 우수 | 순한 포뮬라에 적합 |
| 비이온 (APG 계열) | 우수 | 저자극 프리미엄 라인 |

### 5. 가속 안정성 테스트 조건
- 45°C / 3개월 + 동결-해동 5사이클
- 확인 항목: pH 변화, 외관, 함량 (HPLC)

---

## 실무 포뮬라 예시 (2.5% TXA 젤 클렌저)

```
Phase A (수상)
- 정제수 to 100%
- Tranexamic Acid 2.5%
- Butylene Glycol 5%
- Disodium EDTA 0.05%

Phase B (계면활성제)
- Sodium Cocoyl Isethionate 8%
- Cocamidopropyl Betaine 4%
- Coco-Glucoside 2%

Phase C (점증/보습)
- Hydroxyethylcellulose 0.3%
- Glycerin 3%

Phase D (pH 조절)
- Citric Acid q.s. to pH 5.5
```

## 핵심 체크리스트

- [ ] TXA 완전 용해 확인 (가온 필요 시 60°C 이하)
- [ ] 최종 pH 4.5~6.0 범위
- [ ] 3개월 가속 안정성 통과
- [ ] 계면활성제와 석출/탁도 없음
- [ ] 함량 검증 (HPLC 분석)
