## 1. 최적 농도: 3-5% vs 10%

### 클렌저에서의 권장 농도: **3-5%**

| 항목 | 3-5% | 10% |
|------|------|-----|
| **세정제 효능** | ✅ 충분 | ❌ 과잉/낭비 |
| **자극 위험** | 낮음 | 피부 민감자 자극 가능 |
| **비용 효율** | 높음 | 낮음 |
| **린스오프 특성** | 적합 | 대부분 씻겨나감 |

### 이유
- **접촉 시간이 짧음** (30초~2분): 10% 고농도의 이점이 발휘되기 어려움
- **대부분 씻겨나감**: 농도를 높여도 피부 잔류량 비례 증가 미미
- **Leave-on 제품과 다름**: 세럼/크림은 10%가 효과적이지만, 클렌저는 다른 전략 필요

---

## 2. 피부 잔류율 분석

### 린스오프 제품의 활성 성분 잔류
| 요인 | 영향 |
|------|------|
| **접촉 시간** | 1분 미만 → 침투 제한적 |
| **세정제 계면활성제** | 수용성 성분 용해 → 함께 씻김 |
| **추정 잔류율** | 전체 투입량의 **5-15%** 수준 |

### 실질적 의미
```
클렌저 5% Niacinamide → 피부 잔류 약 0.25-0.75%
클렌저 10% Niacinamide → 피부 잔류 약 0.5-1.5%
```
→ **2배 농도 ≠ 2배 효과** (비용 대비 효율 낮음)

---

## 3. 성분 호환성

### ✅ 호환 가능 (시너지)
| 성분 | 이유 |
|------|------|
| **Hyaluronic Acid** | 보습 시너지, 상호 간섭 없음 |
| **Glycerin** | 보습 강화 |
| **Salicylic Acid (BHA)** | pH 조건 맞으면 병용 가능 |
| **Centella/CICA** | 진정 효과 상승 |
| **Zinc PCA** | 피지 조절 시너지 |

### ⚠️ 주의 필요
| 성분 | 주의사항 |
|------|----------|
| **Vitamin C (L-AA)** | 낮은 pH에서 Niacinamide → Niacin 전환 (홍조 유발) |
| **Strong Acids (AHA/BHA 고농도)** | pH < 3.5에서 불안정 |
| **구리 펩타이드** | 이론상 상호작용 우려 (실제론 미미) |

### pH 안정성
- **최적 pH**: 5.0 - 7.0
- **클렌저 일반 pH**: 5.5 - 6.5 → ✅ 호환 양호
- **pH < 4.0**: Niacinamide가 Niacin으로 분해 → 피부 홍조/자극

## 4. 결론 및 권장사항

### 클렌저 포뮬레이션 권장
```
Niacinamide 농도: 4-5%
pH: 5.5-6.5
추천 병용: Hyaluronic Acid, Glycerin, Zinc PCA
피해야 할 병용: 고농도 Vitamin C, AHA (같은 제품 내)
```

### 마케팅 vs 실효성
| 주장 | 실제 |
|------|------|
| "10% 고농도 나이아신아마이드 클렌저" | 대부분 씻겨나가 5%와 큰 차이 없음 |
| "세안만으로 브라이트닝" | 잔류량 적어 효과 제한적 |

### 전략적 제안
1. **클렌저**: 3-5% (비용 효율 + 자극 최소화)
2. **실질 효과 원하면**: Leave-on 제품(세럼/크림)에 5-10% 추가
3. **더블 포뮬레이션**: 클렌저 4% + 세럼 5% = 총 효과 극대화

---

**요약**: 클렌저에 10%는 과잉이야. **4-5%면 충분**하고, 진짜 효과는 세안 후 남는 세럼/크림에서 나온다고 보면 돼.
