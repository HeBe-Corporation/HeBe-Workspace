## 응답

# Azelaic Acid 클렌저 제형 가이드

## 1. 용해도 문제 해결

Azelaic Acid는 **수용성이 매우 낮음** (25°C에서 약 2.4g/L)

### 해결 전략

| 방법 | 설명 | 장단점 |
|------|------|--------|
| **공용매(Co-solvent)** | Propylene Glycol, Butylene Glycol, Ethoxydiglycol 사용 | 효과적이나 자극 가능성 |
| **알칼리 중화** | Potassium Azelate 형태로 전환 (KOH 사용) | 용해도 ↑, pH 관리 필요 |
| **마이크로에멀젼** | O/W 에멀젼 계면에 분산 | 안정성 우수, 제조 복잡 |
| **나노입자화** | 입자 크기 축소로 분산성 개선 | 비용 높음 |

### 실무 추천 조합
```
Propylene Glycol: 5-10%
Ethoxydiglycol: 3-5%
pH 4.5-5.5 유지 (약산성에서 안정)
```

---

## 2. 클렌저 최적 농도

### 농도별 특성

| 농도 | 효과 | 적합성 |
|------|------|--------|
| 1-2% | 순한 각질케어 | 민감피부, 데일리 |
| 3-5% | 미백 + 피지조절 | **클렌저 권장 범위** |
| 10%+ | 치료 수준 | Leave-on 제품용 |

### 클렌저 특성상 고려사항
- **접촉시간 짧음** (30초-1분) → 높은 농도 필요하지만
- **세정제와 병용** → 피부 자극 증가 가능
- **결론**: **3-5%가 최적**, 민감피부용은 2-3%

---

## 3. 미백 + 여드름 이중 효과 구현

### Azelaic Acid의 이중 작용 메커니즘

```
미백: Tyrosinase 억제 → 멜라닌 합성 ↓
여드름: P. acnes 항균 + 각질세포 정상화 + 항염
```

### 시너지 성분 조합

| 목적 | 추천 성분 | 농도 |
|------|----------|------|
| 미백 부스터 | Niacinamide | 2-4% |
| 여드름 강화 | Salicylic Acid | 0.5-1% |
| 진정/항염 | Centella Asiatica | 0.1-0.5% |
| 보습 보완 | Panthenol | 1-2% |

### 제형 전략

**폼 클렌저 타입**
- 장점: 세정력 우수, 거품으로 균일 분산
- 단점: pH 조절 어려움

**젤 클렌저 타입 (권장)**
- 장점: pH 안정적 유지, 순한 세정
- 적합: Azelaic Acid + Niacinamide 조합

---

## 4. 주의사항

1. **pH 관리**: 4.5-5.5 필수 (너무 낮으면 자극, 높으면 효능 저하)
2. **안정성**: 빛/열에 민감 → 불투명 용기, 서늘한 보관
3. **상호작용**: Vitamin C와 동시 사용 시 pH 충돌 가능
4. **규제 확인**: 국가별 화장품 허용 농도 상이 (한국 OTC 기준 확인 필요)
