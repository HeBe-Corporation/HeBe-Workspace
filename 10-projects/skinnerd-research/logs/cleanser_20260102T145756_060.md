## 응답

# Cotton Candy 클렌저 컨셉 제안

## 1. 텍스처 아이디어

### 메인 추천: 휘핑 무스 (Whipped Mousse)
- **특징**: 솜사탕처럼 가볍고 푹신한 질감
- **사용감**: 피부에 올리면 녹듯이 퍼지며 부드러운 거품 생성
- **기술**: 에어로졸 타입 또는 펌프형 무스

### 대안 옵션
| 텍스처 | 설명 | 장점 |
|--------|------|------|
| 클라우드 폼 | 압축 거품이 피부에서 3배 확장 | 시각적 재미 |
| 젤리 버블 | 투명 젤이 문지르면 거품 전환 | 독특한 경험 |
| 코튼 밤 | 고체가 체온에 녹으며 오일로 변환 | 휴대성 좋음 |

---

## 2. 색상 설계

### 메인 컬러: 파스텔 그라데이션
```
핑크: #FFB6C1 (라이트 핑크) → #FFC0CB (핑크)
블루: #B0E0E6 (파우더 블루) → #ADD8E6 (라이트 블루)
```

### 제품 적용
- **듀얼 챔버**: 핑크/블루가 펌프할 때 섞이며 나옴
- **마블링**: 두 색이 자연스럽게 어우러진 패턴
- **온도 반응**: 차가울 때 블루 → 따뜻해지면 핑크로 변화

### 주의사항
- 천연 착색료 사용 권장 (비트루트 핑크, 스피루리나 블루)
- 염료 알러지 테스트 필수

---

## 3. 향료 구성

### 메인 향: Sweet Cotton Candy Accord
```
탑노트 (5%):    바닐라 + 딸기 + 라즈베리
미들노트 (3%):  캐러멜 + 크림
베이스노트 (2%): 머스크 + 화이트 우드
```

### 향료 옵션
| 버전 | 구성 | 타겟 |
|------|------|------|
| Classic | 전통 솜사탕 (설탕+바닐라) | 향수병 유발, 대중적 |
| Berry Pop | 블루베리+라즈베리 | 상큼함 선호층 |
| Dreamy | 라벤더+바닐라 | 릴렉싱, 저녁용 |

### 기술적 고려
- 잔향 지속: 15-30분 (세안제이므로 가볍게)
- 합성향 최소화, 자연 유래 에센스 블렌딩
- EWG 그린 등급 향료 우선

---

## 4. 패키징 디자인

### 용기 형태
```
[추천] 에어리스 펌프 (150ml)
- 구름/솜사탕 모양 캡
- 반투명 프로스티드 바디 (내용물 색상 비침)
- 그라데이션 인쇄 (핑크→블루)
```

### 소재
| 부위 | 소재 | 이유 |
|------|------|------|
| 바디 | PCR-PET 또는 사탕수수 PE | 친환경 + 반투명 |
| 캡 | 바이오 PP | 구름 조형 가능 |
| 라벨 | 종이 라벨 or 직접 인쇄 | 분리배출 용이 |

### 비주얼 요소
- **일러스트**: 솜사탕 구름 위 토끼/곰 캐릭터
- **타이포**: 둥글고 부드러운 손글씨체
- **홀로그램**: 빛에 따라 무지개빛 반짝임
- **리미티드**: 계절별 색상 변경 (봄 핑크, 여름 민트, 가을 피치, 겨울 라벤더)

### 언박싱 경험
```
1. 외부 박스: 솜사탕 봉지 디자인 (막대 손잡이)
2. 내부 완충재: 분홍 파쇄지 (구름 느낌)
3. 제품 + 미니 거울 또는 파우치 동봉
4. 향기 나는 감사 카드
```

## 5. 마케팅 포인트

### 핵심 메시지
> "세안이 달콤한 추억이 되는 순간"

### USP (Unique Selling Point)
1. **오감 만족**: 보고(색상), 맡고(향), 만지고(텍스처)
2. **SNS 바이럴**: 색다른 비주얼로 공유 유도
3. **무자극 포뮬러**: 민감성 피부 가능

### 타겟
- 1차: 10-20대 여성, K-뷰티/귀여운 제품 선호층
- 2차: 선물용 구매자
