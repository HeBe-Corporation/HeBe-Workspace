# 젤 클렌저 제형 가이드

## 1. 투명도 유지 방법

### 핵심 원칙
- **굴절률 매칭**: 모든 성분의 굴절률을 물(1.33)과 유사하게 맞춤
- **완전 용해**: 불용성 입자나 미용해 성분 제거
- **pH 최적화**: 겔화제별 최적 pH 범위 유지

### 실무 팁
| 요소 | 권장사항 |
|------|----------|
| 계면활성제 | 투명 가용화 가능한 종류 선택 (Cocamidopropyl Betaine 등) |
| 오일 첨가 시 | 가용화제(PEG-40 Hydrogenated Castor Oil) 사용 |
| 향료 | 수용성 또는 가용화 처리된 향료 |
| 온도 | 제조 시 과도한 가열 피함 (기포 발생) |
| 탈기 | 진공 탈기 또는 정치 후 기포 제거 |

## 2. 겔화제 종류별 특성

### 천연/반합성 계열
| 겔화제 | 특성 | 투명도 | 최적 pH |
|--------|------|--------|---------|
| **Carbomer** | 높은 투명도, 전단 희석성 | ★★★★★ | 5.5-7.5 |
| **Xanthan Gum** | 현탁 안정성 우수, 약간 탁함 | ★★★☆☆ | 3-12 |
| **Hydroxyethylcellulose (HEC)** | 투명, 비이온성, 염 안정 | ★★★★☆ | 2-12 |
| **Acrylates/C10-30 Alkyl Acrylate Crosspolymer** | 유화 겔화 가능 | ★★★★☆ | 5-8 |

### 겔화제 선택 기준
```
투명도 우선 → Carbomer 940/980
염 내성 필요 → HEC 또는 Acrylates Copolymer
자연 유래 선호 → Xanthan + HEC 블렌드
```

### Carbomer 중화 시 주의
- **중화제**: Triethanolamine, NaOH, AMP
- **과중화 금지**: pH 7.5 초과 시 점도 급감
- **순서**: 물에 먼저 분산 → 완전 팽윤 → 중화

## 3. 미백 성분 배합 시 주의사항

### 성분별 주의점

| 미백 성분 | 주의사항 | 권장 pH |
|-----------|----------|---------|
| **Niacinamide** | 고온/저pH에서 니코틴산 전환 (홍조 유발) | 5.0-7.0 |
| **Arbutin (α/β)** | 산성 조건에서 하이드로퀴논 유리 가능 | 5.0-7.0 |
| **Ascorbic Acid (비타민C)** | 산화 민감, 금속이온 촉매 | 2.5-3.5 |
| **Tranexamic Acid** | 비교적 안정, 고농도 시 결정화 | 4.5-6.5 |
| **Kojic Acid** | 산화 변색(갈변), 킬레이트제 필요 | 4.0-5.0 |

### 배합 금기/주의 조합
```
❌ Niacinamide + 강산성 환경 (pH < 4)
❌ Arbutin + 강산성 환경
❌ Vitamin C + Niacinamide (같은 제형 내, 논란 있으나 주의)
❌ 금속이온 + 산화 민감 성분
```

### 안정화 전략
1. **항산화제 병용**: Tocopherol, Ferulic Acid
2. **킬레이트제 추가**: EDTA, Phytic Acid
3. **차광 용기**: 갈색/불투명 용기 사용
4. **질소 충전**: 산소 접촉 최소화
5. **적정 농도**: 과량 사용 시 안정성 저하

### 클렌저 특수 고려사항
- 세정 시간이 짧아 미백 효과 제한적
- 고농도보다 **안정성 + 마케팅** 목적이 큼
- 린스오프 제형 특성상 침투 시간 부족
