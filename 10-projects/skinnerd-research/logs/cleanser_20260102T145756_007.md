## 응답

# Kojic Acid 클렌저 사용 시 안정성 및 포뮬레이션 가이드

## 1. 안정성 문제

### 산화 취약성
Kojic acid는 **산소, 빛, 열, 금속 이온**에 매우 민감함:
- 공기 노출 시 빠르게 **갈변(황갈색 변색)**
- pH 의존적 - 알칼리 조건에서 더 불안정
- 금속 이온(Fe, Cu)과 킬레이트 형성 → 변색 가속

### 클렌저 특유의 문제
- 세정제(surfactant)와 상호작용으로 안정성 저하
- 물 함량이 높아 가수분해 위험
- 반복 개봉으로 산소 노출 증가

---

## 2. 변색 방지 방법

### 포뮬레이션 전략

| 방법 | 설명 |
|------|------|
| **pH 조절** | pH 4.0-5.0 유지 (산성 조건에서 안정) |
| **항산화제 첨가** | BHT, Sodium Metabisulfite, Ascorbic Acid |
| **킬레이트제** | EDTA, Phytic Acid로 금속 이온 제거 |
| **Kojic Dipalmitate** | 에스터 유도체 사용 - 훨씬 안정적 |
| **차광 포장** | 불투명 용기, 에어리스 펌프 |
| **질소 충전** | 헤드스페이스 산소 제거 |

### Kojic Acid vs Kojic Dipalmitate

```
Kojic Acid:
- 수용성, 즉각적 효과
- 불안정, 변색 쉬움
- 농도: 1-2%

Kojic Dipalmitate:
- 지용성, 서방형 효과
- 매우 안정적, 변색 없음
- 농도: 2-4%
- 클렌저에 더 적합
```

---

## 3. 다른 미백 성분과의 시너지

### 추천 조합

| 성분 | 시너지 효과 |
|------|------------|
| **Niacinamide (3-5%)** | 멜라닌 전달 억제 + Kojic의 생성 억제 = 상호보완 |
| **Arbutin (1-2%)** | 둘 다 Tyrosinase 억제, 다른 메커니즘 |
| **Vitamin C (MAP)** | 항산화 + 미백, MAP가 안정성 좋음 |
| **Licorice Extract** | 소염 + Glabridin의 미백 효과 |
| **Tranexamic Acid** | 염증성 색소침착에 효과적 |

### 주의할 조합

| 성분 | 주의사항 |
|------|----------|
| **AHA/BHA** | pH 충돌 가능, 자극 증가 |
| **Retinol** | 클렌저에서는 의미 없음 (접촉 시간 짧음) |
| **Hydroquinone** | 규제 성분, 조합 비추천 |

---

## 4. 클렌저에서의 효능 한계

솔직히 말하면:
- 클렌저는 **접촉 시간이 30초-1분** 정도
- Kojic acid가 피부에 흡수될 시간이 부족
- **세럼이나 크림**에서 효과가 훨씬 좋음

### 클렌저에 넣는 이유
- 마케팅 포인트
- 미백 루틴의 "첫 단계" 느낌
- 실제 효과는 leave-on 제품 대비 **10-20%** 수준

---

## 5. 실용적 권장사항

클렌저 포뮬레이션 시:
1. **Kojic Dipalmitate 2-3%** 사용 (안정성 확보)
2. **Niacinamide 2%** 추가 (시너지)
3. **pH 4.5** 맞추기
4. **EDTA 0.1%** 첨가
5. **불투명 튜브** 포장

진정한 미백 효과 원하면 **leave-on 제품**(세럼, 앰플)에 집중하는 게 더 효율적임.
