## 응답

# Glutathione 클렌저 제형 분석

## 1. 최적 농도

| 용도 | 농도 범위 | 비고 |
|------|-----------|------|
| 일반 미백 효과 | 0.5-1.0% | 일상 사용에 적합 |
| 집중 미백 | 1.0-2.0% | 단기간 집중 케어 |
| 클렌저 권장 | **0.1-0.5%** | 짧은 접촉 시간 고려 |

클렌저는 피부 접촉 시간이 30초~2분으로 짧아서, 세럼/크림 대비 **낮은 농도로도 충분**함. 고농도는 비용 대비 효과가 떨어짐.

---

## 2. 안정화 방법

Glutathione(GSH)은 **산화에 매우 취약**함. 공기/빛/열에 노출되면 GSSG(산화형)로 변환되어 효능 상실.

### 주요 안정화 전략

| 방법 | 설명 |
|------|------|
| **캡슐화** | 리포좀, 사이클로덱스트린 포접 |
| **산화방지제 병용** | Vitamin C, E, Ferulic Acid 추가 |
| **킬레이트제** | EDTA, Phytic Acid로 금속이온 제거 |
| **불활성 가스 충진** | 질소 퍼징으로 산소 제거 |
| **차광 용기** | 불투명/갈색 용기 사용 |
| **pH 조절** | 산성 pH 유지 (아래 참조) |

### 유도체 사용
순수 GSH 대신 **안정화 유도체** 사용:
- **Glutathione Liposome** - 흡수율/안정성 향상
- **S-Acetyl Glutathione** - 지용성, 세포 투과 우수
- **Sodium Glutathione** - 수용성, 제형 용이

---

## 3. pH 조건

| pH 범위 | 상태 |
|---------|------|
| **pH 3.5-5.5** | 최적 안정 구간 |
| pH 5.0-6.0 | 피부 친화성 우수 |
| pH 7.0 이상 | 급격한 분해 |

**클렌저 권장 pH: 4.5-5.5**
- GSH 안정성 확보
- 피부 장벽(pH 4.5-5.5)과 호환
- 세정력 유지

---

## 4. 클렌저에서의 효능 발현 메커니즘

### 미백 경로

```
Glutathione
    │
    ├─① Tyrosinase 직접 억제
    │     └→ 멜라닌 생성 감소
    │
    ├─② Pheomelanin 전환 촉진
    │     └→ 검은색(Eumelanin) → 밝은색(Pheomelanin)
    │
    ├─③ 항산화 작용
    │     └→ 자유라디칼 제거 → 멜라닌 과생성 억제
    │
    └─④ 각질층 침투
          └→ 세안 중 표피층에 일부 흡착
```

### 클렌저 특성상 한계

| 요소 | 설명 |
|------|------|
| **접촉 시간** | 30초~2분, 충분한 침투 어려움 |
| **계면활성제** | GSH 안정성 저하 가능 |
| **헹굼** | 대부분 씻겨나감 |

### 실질적 기대 효과

1. **즉각적 브라이트닝** - 각질층 표면 정리
2. **프라이밍 효과** - 다음 스킨케어 흡수 촉진
3. **누적 효과** - 장기 사용 시 미백 보조
4. **항산화 부스팅** - 세안 중 산화 스트레스 감소

---

## 5. 제형 설계 권장사항

```
[권장 클렌저 조성]
├─ Glutathione: 0.3-0.5%
├─ Niacinamide: 2-3% (시너지)
├─ Vitamin C 유도체: 0.5-1%
├─ pH: 5.0 ± 0.3
├─ 킬레이트제: EDTA 0.05%
└─ 차광 튜브 포장
```

---

## 결론

클렌저에서 Glutathione은 **보조적 미백 성분**으로 접근해야 함. 주력 미백 효과는 세럼/크림에서 기대하고, 클렌저는 **프라이밍 + 누적 효과** 목적으로 0.3-0.5% 농도, pH 4.5-5.5 조건에서 안정화 유도체를 사용하는 것이 합리적임.
