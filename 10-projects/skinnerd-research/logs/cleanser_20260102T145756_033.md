## 응답

# 클렌저 원가 구조 분석

## 1. 일반적인 원가 구조 (제조원가 기준)

| 항목 | 비중 | 설명 |
|------|------|------|
| **원료비** | 15-30% | 계면활성제, 보습제, 기능성 원료 |
| **용기/패키징** | 20-35% | 튜브, 펌프, 박스, 라벨 |
| **인건비** | 5-10% | 제조 인력, 품질관리 |
| **제조간접비** | 5-10% | 설비, 유틸리티, 감가상각 |
| **OEM/ODM 마진** | 10-20% | 위탁생산 시 |

## 2. 유통 단계별 마진 구조

```
제조원가 100원 기준

제조사 출고가: 130-150원 (30-50% 마진)
     ↓
도매/유통: 180-220원 (30-50% 마진)
     ↓
소매가: 300-400원 (50-80% 마진)
```

**실제 예시 (15,000원 클렌저)**
| 단계 | 금액 | 비중 |
|------|------|------|
| 제조원가 | 3,000-4,500원 | 20-30% |
| 브랜드 마진 | 3,000-4,500원 | 20-30% |
| 유통 마진 | 4,000-6,000원 | 27-40% |
| 마케팅비 | 1,500-3,000원 | 10-20% |

## 3. 가격대별 원료 선택 전략

### 저가 (5,000원 이하)
| 원료 유형 | 선택 전략 |
|-----------|-----------|
| 계면활성제 | SLS, SLES (저렴, 세정력 강) |
| 보습제 | 글리세린 (기본) |
| 기능성 | 최소화 또는 미량 첨가 |
| 용기 | 플라스틱 튜브, 단순 디자인 |

### 중가 (10,000-20,000원)
| 원료 유형 | 선택 전략 |
|-----------|-----------|
| 계면활성제 | 코코베타인, 아미노산계 일부 혼합 |
| 보습제 | 히알루론산, 판테놀 추가 |
| 기능성 | 녹차/티트리 등 천연추출물 |
| 용기 | 펌프형, 에어리스 일부 |

### 고가 (30,000원 이상)
| 원료 유형 | 선택 전략 |
|-----------|-----------|
| 계면활성제 | 100% 아미노산계 (글루타민산, 글리신계) |
| 보습제 | 세라마이드, 스쿠알란, 펩타이드 |
| 기능성 | 고농도 활성원료, 특허 성분 |
| 용기 | 에어리스, 유리, 고급 패키징 |

## 4. 주요 원료 단가 참고

| 원료 | kg당 가격 | 비고 |
|------|-----------|------|
| SLS/SLES | 3,000-5,000원 | 가장 저렴 |
| 코코베타인 | 8,000-12,000원 | 중간 |
| 아미노산계 | 30,000-80,000원 | 고가 |
| 글리세린 | 3,000-5,000원 | 기본 보습 |
| 히알루론산 | 50,000-200,000원 | 등급별 차이 |
| 세라마이드 | 100,000-500,000원 | 프리미엄 |

## 5. 수익성 최적화 포인트

### 원가 절감
- 용기 대량 발주 (MOQ 높이면 20-30% 절감)
- 원료 직접 소싱 (중간 유통 제거)
- 제형 단순화 (SKU 줄이기)

### 마진 확보
- D2C 채널 강화 (유통 마진 흡수)
- 번들/정기구독 (객단가 상승)
- 리필 시스템 (용기비 절감 + 재구매 유도)
