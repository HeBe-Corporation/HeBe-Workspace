## 응답

# 버블 클렌저 vs 폼 클렌저 비교 분석

## 기본 정의

| 구분 | 버블 클렌저 | 폼 클렌저 |
|------|------------|----------|
| **형태** | 펌프 시 이미 거품 상태로 나옴 | 크림/젤 형태 → 직접 거품 생성 |
| **용기** | 폼 펌프 용기 (메쉬 내장) | 일반 튜브/펌프 |

---

## 거품 생성 메커니즘

### 버블 클렌저
- **펌프 구조**: 메쉬 필터 + 공기 혼합 챔버
- **원리**: 액상 제형이 펌프질할 때 메쉬를 통과하며 공기와 강제 혼합
- **계면활성제**: 저농도 (2-5%)로도 거품 생성 가능
- **거품 특성**: 가볍고 부드러운 마이크로 버블

### 폼 클렌저
- **원리**: 계면활성제가 물과 마찰로 공기를 포집
- **계면활성제**: 고농도 필요 (8-15%)
- **거품 특성**: 밀도 높고 탄탄한 거품 (사용자 기술에 따라 다름)

---

## 세정력 비교

| 항목 | 버블 클렌저 | 폼 클렌저 |
|------|------------|----------|
| **메이크업** | △ 가벼운 메이크업 | ○ 중간~진한 메이크업 |
| **피지 제거** | △ 보통 | ○ 우수 |
| **노폐물** | ○ 일반 노폐물 | ○ 모공 속 노폐물 |
| **선크림** | △ 워터프루프는 어려움 | ○ 대부분 제거 가능 |

**핵심 차이**: 계면활성제 농도 차이 → 폼 클렌저가 세정력 우위

---

## 피부 자극도

### 버블 클렌저 (저자극)
- 계면활성제 농도 낮음
- pH 5.5~6.5 약산성 제형 많음
- 세정 후 당김 적음
- **적합**: 민감성, 건성, 아토피 피부

### 폼 클렌저 (중간~강자극)
- 계면활성제 농도 높음
- pH 7~9 약알칼리성도 존재
- 세정 후 뽀득함/당김 가능
- **적합**: 지성, 복합성, 트러블 피부

---

## 장단점 정리

### 버블 클렌저

| 장점 | 단점 |
|------|------|
| 즉시 거품 → 편리함 | 세정력 부족 (진한 메이크업 X) |
| 저자극 | 가성비 낮음 (용량 대비 빨리 소진) |
| 거품망 불필요 | 펌프 고장 시 사용 불가 |
| 아이/민감피부 적합 | 워터프루프 제거 어려움 |

### 폼 클렌저

| 장점 | 단점 |
|------|------|
| 강력한 세정력 | 피부 자극 가능성 |
| 가성비 좋음 | 거품 생성 번거로움 |
| 다양한 제형 선택 | 과세정 주의 필요 |
| 휴대 용이 | 사용량 조절 실패 가능 |

---

## 선택 가이드

```
민감성/건성/아토피 → 버블 클렌저
지성/복합성/트러블 → 폼 클렌저
가벼운 메이크업 → 버블 클렌저
풀 메이크업 → 폼 클렌저 (또는 더블 클렌징)
아침 세안 → 버블 클렌저
저녁 세안 → 폼 클렌저
```

---

## 결론

**버블 클렌저**: 편의성 + 저자극 우선
**폼 클렌저**: 세정력 + 가성비 우선

피부 타입과 메이크업 정도에 따라 선택하거나, 아침(버블) / 저녁(폼) 조합 사용 추천.
