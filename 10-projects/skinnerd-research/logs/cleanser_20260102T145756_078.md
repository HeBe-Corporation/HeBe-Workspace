## 질문
Licorice Root Extract(감초추출물)의 미백 성분(Glabridin, Liquiritin)별 효능 차이와 클렌저 적용 시 권장 농도를 알려줘.

## 응답

# Licorice Root Extract 미백 성분 비교

## 주요 미백 성분별 효능 차이

### 1. Glabridin (글라브리딘)
- **작용 기전**: 티로시나제 억제 (IC50: 0.1-0.5 μM)
- **특징**: 
  - 가장 강력한 미백 효과 (하이드로퀴논 대비 16배)
  - 항염 + 항산화 효과 동시 보유
  - 지용성 → 피부 침투력 우수
- **단점**: 추출 수율 낮음, 고가

### 2. Liquiritin (리퀴리틴)
- **작용 기전**: 멜라닌 분산 촉진 + 티로시나제 억제
- **특징**:
  - 수용성 → 제형화 용이
  - 기존 멜라닌 분해 효과
  - 가격 대비 효율 좋음
- **단점**: Glabridin 대비 효능 약함 (약 1/10)

### 비교 요약

| 항목 | Glabridin | Liquiritin |
|------|-----------|------------|
| 용해성 | 지용성 | 수용성 |
| 티로시나제 억제력 | ★★★★★ | ★★★ |
| 멜라닌 분해 | ✗ | ✓ |
| 침투력 | 높음 | 보통 |
| 가격 | 고가 | 중가 |
| 안정성 | 보통 | 우수 |

## 클렌저 적용 시 권장 농도

### 클렌저 특성 고려사항
- **짧은 접촉 시간** (30초~2분)
- **세정 후 씻어냄** → 유효 성분 잔류 최소화
- **미백 효과보다 마케팅/부가 효과** 목적

### 권장 농도

| 성분 | 권장 농도 | 비고 |
|------|-----------|------|
| Licorice Extract (일반) | 0.5-2.0% | 표준화 안 된 추출물 |
| Glabridin 함유 추출물 | 0.1-0.5% | Glabridin 기준 40% 표준화 시 |
| Liquiritin 함유 추출물 | 0.5-1.0% | |
| 순수 Glabridin | 0.01-0.05% | 고가, 클렌저엔 비효율 |

### 실무 권장
```
클렌저 제형: Licorice Root Extract 1.0% 
(Glabridin 40% 표준화 제품 기준)

목적: 항염/진정 효과 + 미백 클레임
실제 미백 효과: 제한적 (wash-off 제형 한계)
```

## 결론

클렌저에서 감초추출물은 **실질적 미백보다 진정/항염 효과**가 주 목적. 진정한 미백 효과를 원하면 **세럼/크림** 등 leave-on 제형에 **Glabridin 0.1% 이상** 배합 권장.
