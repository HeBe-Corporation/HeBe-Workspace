## 응답

# 미백 성분 조합 최적화: Glutathione + Niacinamide + Arbutin

## 시너지 효과 메커니즘

세 성분은 **멜라닌 생성 경로의 서로 다른 단계**를 타겟팅해 시너지 발생:

| 성분 | 작용 메커니즘 | 타겟 단계 |
|------|--------------|----------|
| **Glutathione** | 티로시나제 억제 + 유멜라닌→페오멜라닌 전환 유도 | 멜라닌 합성 초기 |
| **Niacinamide** | 멜라노좀의 케라티노사이트 전달 차단 | 멜라닌 이동 |
| **Arbutin** | 티로시나제 활성 억제 (하이드로퀴논 유도체) | 멜라닌 합성 중기 |

```
[티로신] → [DOPA] → [도파퀴논] → [멜라닌] → [피부 표면]
    ↑           ↑                         ↑
 Glutathione  Arbutin              Niacinamide
  (억제)       (억제)                 (전달 차단)
```

## 권장 배합 비율

### 기본 포뮬러 (수용성 에센스 기준)

| 성분 | 권장 농도 | 유효 범위 | 비고 |
|------|----------|----------|------|
| **Niacinamide** | 4-5% | 2-10% | 5% 초과 시 홍조 가능 |
| **Alpha-Arbutin** | 1-2% | 0.5-2% | Beta보다 10배 효과적 |
| **Glutathione** | 0.5-1% | 0.1-2% | 안정성 이슈로 낮게 |

### 비율 예시 (총 활성성분 기준)
```
Niacinamide : Alpha-Arbutin : Glutathione
    5       :      2        :     1
```

---

## 제형별 권장 조합

### 1. 데일리 에센스 (순한 버전)
- Niacinamide 4%
- Alpha-Arbutin 1%
- Glutathione 0.5%
- pH 5.5-6.5

### 2. 인텐시브 세럼 (집중 케어)
- Niacinamide 5%
- Alpha-Arbutin 2%
- Glutathione 1%
- Vitamin C (AA2G) 2% 추가 가능
- pH 5.0-6.0

### 3. 나이트 크림
- Niacinamide 3%
- Alpha-Arbutin 1.5%
- Glutathione 0.5%
- Retinol 0.1% 병용 시 주의

---

## 안정성 고려사항

| 요소 | 권장 |
|------|------|
| **pH** | 5.0-7.0 (Arbutin은 pH 5-7에서 안정) |
| **온도** | 25°C 이하 보관 |
| **광 차단** | 갈색/불투명 용기 필수 |
| **산화 방지** | Glutathione은 산화 민감 → 진공 펌프 용기 권장 |
| **상호작용** | Vitamin C(LAA)와 Niacinamide 직접 혼합 피하기 |

## 사용 권장

- **도포 순서**: 토너 → 에센스(본 조합) → 크림 → 자외선차단제
- **사용 빈도**: 아침/저녁 2회, 민감피부는 저녁 1회로 시작
- **효과 발현**: 4-8주 지속 사용 시 가시적 효과
- **필수 병용**: SPF 30+ 자외선차단제 (미백 효과 유지 필수)

---

## 주의사항

1. **Arbutin 순도**: 불순물(하이드로퀴논) 0.1% 미만 확인
2. **Glutathione 형태**: L-Glutathione(환원형) 사용, 산화형은 효과 없음
3. **Niacinamide 순도**: 니코틴산 불순물 시 홍조 유발

추가로 원료 소싱이나 안정화 테스트 방법이 필요하면 알려줘.
