# 클렌저 OEM 품질 관리 체크포인트

## 1. 미생물 테스트

| 항목 | 기준 | 비고 |
|------|------|------|
| 총호기성생균수 | ≤1,000 CFU/g | 일반 화장품 기준 |
| 진균(곰팡이/효모) | ≤100 CFU/g | |
| 대장균 | 불검출 | 필수 |
| 녹농균 | 불검출 | 필수 |
| 황색포도상구균 | 불검출 | 필수 |
| 방부력 테스트 (Challenge Test) | 14~28일 관찰 | 접종 후 균 감소율 확인 |

## 2. 안정성 테스트

| 항목 | 조건 | 확인 사항 |
|------|------|----------|
| 고온 안정성 | 45°C, 4주 | 변색, 분리, 침전, pH 변화 |
| 저온 안정성 | 4°C, 4주 | 결정화, 겔화, 점도 변화 |
| 온도순환 | -10°C↔45°C 반복 | 유화 깨짐, 층분리 |
| 광안정성 | 자외선 조사 (ICH 가이드) | 변색, 성분 분해 |
| 장기보존 | 상온 25°C, 12~24개월 | 사용기한 설정 근거 |
| 원심분리 | 3,000rpm, 30분 | 즉시 분리 여부 확인 |

## 3. 사용감 테스트

| 항목 | 평가 방법 | 체크 포인트 |
|------|----------|------------|
| 세정력 | 인공피지 제거율 측정 | 메이크업/피지 제거 정도 |
| 거품력 | Ross-Miles 법 또는 관능평가 | 거품량, 거품질감, 지속시간 |
| 헹굼성 | 물 헹굼 후 잔여감 | 미끌거림, 당김 |
| 피부자극 | 패치테스트 (48시간) | 홍반, 가려움 |
| 보습감 | 세안 후 수분량 측정 | Corneometer 측정 |
| 향취 | 관능평가 패널 | 강도, 지속성, 선호도 |
| 사용편의성 | 펌프 토출량, 텍스처 | 점도, 발림성 |

## 4. 이화학 테스트

| 항목 | 측정 | 허용범위 |
|------|------|----------|
| pH | pH 미터 | 4.5~7.0 (피부 친화) |
| 점도 | 점도계 | 규격 ±10% |
| 비중 | 비중계 | 규격 ±5% |
| 색상 | 색차계 | ΔE ≤3.0 |
| 향취 | 관능 비교 | 표준품 대비 동등 |
