## 응답

# Kojic Acid 클렌저 사용 시 안정성 및 시너지

## 1. 안정성 문제

Kojic Acid는 클렌저에서 **가장 불안정한 미백 성분 중 하나**:

| 문제 | 원인 | 결과 |
|------|------|------|
| **산화** | 공기/빛 노출 | 갈색 변색 |
| **pH 민감성** | pH 5 이상에서 급격히 분해 | 효능 상실 |
| **금속 이온 반응** | Fe²⁺, Cu²⁺와 킬레이트 | 검은색 변색 |
| **열 불안정** | 고온 제조 공정 | 분해 가속화 |

클렌저는 특히 문제가 되는 이유:
- 물 함량 높음 → 가수분해 촉진
- 계면활성제가 산화 가속화
- 사용 시 공기 노출 빈번

---

## 2. 변색 방지 방법

### 제형 안정화
```
1. pH 조절: 4.0~5.0 유지 (최적 pH 4.5)
2. 킬레이트제 첨가: EDTA, Phytic Acid
3. 항산화제 병용: Sodium Metabisulfite, BHT
4. 불투명 용기: 차광 패키징 필수
```

### 대안: Kojic Acid Dipalmitate
- 지용성 유도체
- 산화 안정성 10배 이상 향상
- 피부 침투 후 Kojic Acid로 전환
- 변색 문제 거의 없음

### 제조 시 주의사항
- 저온 공정 (40°C 이하)
- 질소 퍼징으로 산소 제거
- 스테인리스 또는 유리 용기 사용 (금속 이온 오염 방지)

---

## 3. 다른 미백 성분과의 시너지

### 추천 조합

| 조합 성분 | 시너지 효과 | 비고 |
|-----------|-------------|------|
| **Arbutin** | Tyrosinase 이중 억제 | 가장 흔한 조합 |
| **Niacinamide** | 멜라닌 전달 차단 + 생성 억제 | pH 주의 (6~7 선호) |
| **Vitamin C (AA2G)** | 항산화 + 미백 | 안정화된 유도체 사용 |
| **Licorice Extract** | 항염 + Tyrosinase 억제 | 자극 완화 효과 |
| **Tranexamic Acid** | 멜라닌 생성 신호 차단 | 수용성, 혼합 용이 |

### 피해야 할 조합

| 성분 | 문제 |
|------|------|
| **AHA/BHA (고농도)** | pH 충돌, 자극 증가 |
| **Retinol** | 산화 촉진, 자극 |
| **Benzoyl Peroxide** | 산화제와 직접 반응 |

---

## 4. 클렌저에서의 실용적 고려

### 효능 한계
클렌저는 **접촉 시간 30초~1분** → 미백 효과 제한적

### 권장 전략
```
클렌저: 0.5~1% Kojic Acid (프라이밍 역할)
    ↓
세럼/크림: 2% Kojic Acid + Arbutin (메인 효과)
```

### 대안 제안
클렌저보다 **Leave-on 제품**이 Kojic Acid에 더 적합:
- 토너, 세럼, 크림
- 접촉 시간 확보
- 안정화 더 용이
