## 응답

# PDRN 클렌저 적용 분석

## 1. 분자량 고려사항

**PDRN (Polydeoxyribonucleotide) 특성:**
- 분자량: 50-1,500 kDa (평균 ~270 kDa)
- 연어/송어 정자 DNA 유래 고분자 폴리머

**클렌저 적용 시 문제점:**
| 요소 | 상세 |
|------|------|
| 피부 침투 | 500 Da 이상은 각질층 통과 어려움 → PDRN은 침투 불가 |
| 수용성 | 높은 수용성 → 물에 쉽게 헹궈짐 |
| 계면활성제 영향 | 세정제와 혼합 시 분자 구조 변성 가능 |

## 2. 농도 분석 (0.01-0.1%)

**비교 기준:**
- 주사제/메조테라피: 0.5-1% (주입으로 진피 도달)
- 세럼/앰플: 0.1-1% (도포 후 흡수 시간 확보)
- 클렌저: 0.01-0.1% (접촉 시간 30초-2분)

**클렌저 농도의 현실:**
```
0.01% = 100ppm = 1L당 0.1g
0.1%  = 1000ppm = 1L당 1g
```

→ 세럼 대비 1/10~1/100 농도에서 더 짧은 접촉 시간

## 3. 세정 시간 내 효능 발현 가능성

**PDRN 작용 메커니즘:**
1. A2A 아데노신 수용체 활성화 → 항염, 조직재생
2. DNA 합성 촉진 → 세포 증식

**시간 요구량 vs 클렌저 접촉 시간:**
| 단계 | 필요 시간 | 클렌저 접촉 |
|------|-----------|-------------|
| 피부 흡착 | 수초 | ✓ 가능 |
| 각질층 침투 | 수분-수시간 | ✗ 불가 |
| 수용체 결합 | 수분 이상 | ✗ 불가 |
| 세포 신호전달 | 수시간 | ✗ 불가 |

## 결론

| 항목 | 평가 |
|------|------|
| 과학적 효능 | **매우 낮음** - 분자량, 접촉시간 모두 부적합 |
| 마케팅 가치 | 있음 - "PDRN 함유" 소구 가능 |
| 실질 기능 | 보습제 역할 정도 (헹굼 전까지) |

**권장 대안:**
1. **저분자 뉴클레오타이드** - PDRN 가수분해물 (< 500 Da)
2. **Leave-on 제품에 적용** - 세럼, 크림, 마스크팩
3. **다른 기능성 성분** - 클렌저엔 저분자 보습제(글리세린, 히알루론산 올리고머)가 더 적합

---

**요약:** PDRN의 고분자 특성상 클렌저(30초-2분 접촉, 헹굼)에서 생물학적 활성을 기대하기 어려움. 마케팅 소구 외 실질 효능은 제한적.
