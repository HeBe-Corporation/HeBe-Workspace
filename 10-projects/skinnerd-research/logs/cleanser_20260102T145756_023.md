# 밀크 클렌저 제형 설계 가이드

## 1. 유화 안정성 확보 방법

### O/W 유화 시스템 기본 원칙
- **유화제 조합**: 주유화제(HLB 12-16) + 보조유화제(HLB 4-6) 병용
- **상전이 온도**: 유상과 수상을 70-75°C에서 혼합
- **균질화**: 고속 믹서(3000-5000 rpm) 또는 호모게나이저 사용
- **냉각 속도**: 급냉 피하고 서서히 40°C까지 냉각

### 안정화 전략
| 방법 | 성분 예시 | 역할 |
|------|----------|------|
| 점도 조절 | Xanthan Gum, Carbomer | 크리밍/분리 방지 |
| 계면 강화 | Cetyl Alcohol, Stearic Acid | 유화막 강화 |
| 전해질 조절 | NaCl 최소화 | 이온성 유화제 안정성 유지 |

## 2. HLB 밸런스

### 밀크 클렌저 적정 HLB
- **목표 HLB**: 10-14 (O/W 시스템)
- **오일상 Required HLB에 맞춤**

### 주요 오일별 Required HLB
| 오일 | Required HLB |
|------|-------------|
| Mineral Oil | 10-12 |
| Isopropyl Myristate | 11.5 |
| Jojoba Oil | 6-7 |
| Squalane | 10-11 |

### HLB 계산식
```
혼합 HLB = (HLB_A × 중량_A + HLB_B × 중량_B) / (중량_A + 중량_B)
```

### 추천 유화제 조합
```
예시: Polysorbate 60 (HLB 14.9) + Sorbitan Stearate (HLB 4.7)
      → 비율 조절로 목표 HLB 달성
```

## 3. 미백 성분 배합 전략

### 클렌저에서의 한계
- **접촉 시간 짧음** (30초-1분)
- **세정 후 씻겨나감**
- **고농도 배합 어려움**

### 현실적 접근
| 전략 | 성분 | 비고 |
|------|------|------|
| 마일드 미백제 | Niacinamide 2-4% | 안정적, 자극 적음 |
| 각질케어 병행 | Lactic Acid 0.5-1% | 칙칙함 제거 |
| 항산화 | Vitamin C 유도체 | MAP, SAP (안정형) |
| 감초추출물 | Glabridin | 자연유래 |

### 배합 시 주의점
1. **pH 조절**: Niacinamide는 pH 5-7 안정
2. **Vitamin C 유도체**: 열/빛 민감 → 냉각 후 투입
3. **기능성 인정**: 미백 표시하려면 식약처 인증 필요

### 추천 기본 처방 골격
```
수상 (80%):
  정제수, Glycerin 5%, Niacinamide 3%
  
유상 (15%):
  Mineral Oil 10%, Cetyl Alcohol 2%
  Polysorbate 60 2%, Sorbitan Stearate 1%
  
첨가상 (5%):
  방부제, 향료, 점도조절제
```
