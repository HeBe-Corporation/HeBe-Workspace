# 동남아 미백 화장품 규제 요약

## 1. ASEAN 공통 규정 (ASEAN Cosmetic Directive)

ASEAN 10개국은 **ASEAN Cosmetic Directive (ACD)**를 기반으로 화장품을 규제함.

### 금지 성분 (전면 사용 금지)
| 성분 | 이유 |
|------|------|
| **Hydroquinone** (2% 초과) | 피부 손상, 발암 우려 |
| **Mercury (수은)** 및 화합물 | 신경독성, 신장 손상 |
| **Lead acetate** | 중금속 독성 |
| **Corticosteroids** | 피부 위축, 호르몬 교란 |

### 제한 허용 성분
| 성분 | 최대 허용 농도 |
|------|---------------|
| Arbutin (α, β) | **7%** |
| Kojic acid | **1~2%** (국가별 상이) |
| Niacinamide | **5%** (일반), 의약외품은 상향 가능 |
| Ascorbic acid (비타민C) | 제한 없음 (안정성 문제로 유도체 선호) |
| Tranexamic acid | **2~3%** |

## 2. 국가별 상세 규정

### 🇻🇳 베트남 (Vietnam)

**관할**: 보건부 (Ministry of Health) - Drug Administration of Vietnam (DAV)

| 구분 | 내용 |
|------|------|
| **금지** | Hydroquinone (OTC 전면 금지), Mercury, Tretinoin |
| **허용** | Arbutin ≤7%, Kojic acid ≤1%, Niacinamide ≤5% |
| **필수 표기** | 베트남어 라벨, 성분 전성분 표기, 제조일/유통기한, 수입허가번호 |
| **특이사항** | "미백(làm trắng)" 효능 표방 시 기능성 화장품 등록 필요 |

---

### 🇹🇭 태국 (Thailand)

**관할**: Thai FDA (Food and Drug Administration)

| 구분 | 내용 |
|------|------|
| **금지** | Hydroquinone (2% 초과), Mercury, Clobetasol |
| **허용** | Arbutin ≤7%, Kojic acid ≤2%, Niacinamide 제한 없음 |
| **필수 표기** | 태국어 라벨, FDA 등록번호, 전성분(INCI), 제조국 |
| **특이사항** | "Whitening/미백" 클레임 시 **특수화장품** 분류, 추가 서류 필요 |

---

### 🇰🇭 캄보디아 (Cambodia)

**관할**: Ministry of Health - Department of Drugs and Food

| 구분 | 내용 |
|------|------|
| **금지** | ASEAN 공통 금지 성분 준용 (Hydroquinone, Mercury 등) |
| **허용** | ASEAN Cosmetic Directive 기준 적용 |
| **필수 표기** | 크메르어 또는 영어 라벨, 수입허가번호, 유통기한 |
| **특이사항** | 규제 집행이 상대적으로 느슨, 시장 모니터링 중요 |

---

### 🇮🇩 인도네시아 (Indonesia)

**관할**: BPOM (Badan Pengawas Obat dan Makanan)

| 구분 | 내용 |
|------|------|
| **금지** | Hydroquinone (OTC 금지), Mercury, Retinoic acid |
| **허용** | Arbutin ≤7%, Kojic acid ≤1%, Niacinamide ≤4% |
| **필수 표기** | 인도네시아어 라벨, BPOM 등록번호, 할랄 인증 (선호) |
| **특이사항** | **할랄 인증** 사실상 필수 (2026년 의무화 예정), 미백 클레임 엄격 심사 |

---

### 🇲🇾 말레이시아 (Malaysia)

**관할**: NPRA (National Pharmaceutical Regulatory Agency)

| 구분 | 내용 |
|------|------|
| **금지** | Hydroquinone (2% 초과), Mercury, Tretinoin |
| **허용** | Arbutin ≤7%, Kojic acid ≤2%, Tranexamic acid ≤2% |
| **필수 표기** | 말레이어/영어 라벨, NOT 번호, 전성분 |
| **특이사항** | 미백 제품 적발 사례 많음 (불법 스테로이드 혼입), 시장 감시 강화 중 |

---

### 🇵🇭 필리핀 (Philippines)

**관할**: FDA Philippines

| 구분 | 내용 |
|------|------|
| **금지** | Hydroquinone (OTC 2% 초과), Mercury, Clobetasol |
| **허용** | Arbutin ≤7%, Kojic acid ≤2%, Glutathione (경구/주사 규제 중) |
| **필수 표기** | 영어/필리핀어 라벨, FDA 등록번호, 전성분 |
| **특이사항** | Glutathione 주사제 금지, 화장품 내 사용은 허용 |

---

### 🇸🇬 싱가포르 (Singapore)

**관할**: HSA (Health Sciences Authority)

| 구분 | 내용 |
|------|------|
| **금지** | Hydroquinone (2% 초과, 처방 필요), Mercury |
| **허용** | Arbutin ≤7%, Kojic acid 제한 없음, Niacinamide 제한 없음 |
| **필수 표기** | 영어 라벨, 전성분(INCI), 제조사/수입사 정보 |
| **특이사항** | 규제 가장 엄격, 허위 광고 시 과징금 |

## 3. 공통 필수 표기사항

| 항목 | 세부 |
|------|------|
| 제품명 | 현지어/영어 |
| 전성분 (INCI) | 함량 순 |
| 용량/중량 | 단위 표기 |
| 제조일/유통기한 | 또는 개봉 후 사용기한 (PAO) |
| 제조국 | Made in XXX |
| 제조사/수입사 | 주소, 연락처 |
| 등록번호 | 국가별 형식 상이 |
| 사용법/주의사항 | 알레르기, 패치테스트 권고 등 |

---

## 4. 주의사항 (수출 시)

1. **Hydroquinone**: 대부분 OTC 금지, 처방 의약품으로만 허용
2. **할랄 인증**: 인도네시아, 말레이시아는 사실상 필수
3. **미백 클레임**: "whitening", "lightening", "brightening" 용어 사용 시 기능성/특수 화장품 분류될 수 있음
4. **현지어 라벨**: 대부분 현지어 표기 의무화
