# 클렌저 pH 최적화: 미백 성분별 전략

## 피부 pH 기본 개념

**건강한 피부 pH**: 4.5~5.5 (약산성)
- 산성 맨틀 유지 → 세균/진균 방어
- 장벽 기능 최적화
- 효소 활성 유지

---

## 미백 성분별 최적 pH 범위

| 성분 | 최적 pH | 클렌저 적용 가능성 | 비고 |
|------|---------|-------------------|------|
| **비타민 C (L-AA)** | 2.5~3.5 | △ 낮음 | 너무 산성, 자극 우려 |
| **나이아신아마이드** | 5.0~7.0 | ◎ 최적 | 피부 pH와 호환 좋음 |
| **알부틴** | 5.0~7.0 | ◎ 최적 | 안정성 좋음 |
| **코직산** | 4.0~5.0 | ○ 양호 | 피부 pH 범위 내 |
| **아젤라산** | 4.0~5.0 | ○ 양호 | 약산성 호환 |
| **트라넥삼산** | 6.0~8.0 | △ 보통 | 중성~약알칼리 선호 |
| **감초추출물** | 4.5~6.5 | ◎ 최적 | 넓은 pH 범위 |
| **AHA (글리콜산)** | 3.0~4.0 | △ 낮음 | 필링 효과, 자극 주의 |

## 클렌저 pH 밸런스 전략

### 1. 권장 클렌저 pH 범위: **5.0~6.0**

```
이유:
- 피부 pH(4.5-5.5)에서 크게 벗어나지 않음
- 세정력 확보 (너무 산성이면 세정력↓)
- 대부분 미백 성분 안정성 유지
```

### 2. 성분 조합 전략

**Best 조합 (pH 5.0~5.5)**
```
나이아신아마이드 + 알부틴 + 감초추출물
→ 자극 최소, 안정성 최대
```

**주의 조합**
```
비타민C + 나이아신아마이드
→ pH 차이 큼, 클렌저보다 세럼에서 분리 사용 권장
```

### 3. 제형별 접근

| 제형 | 권장 pH | 적합 성분 |
|------|---------|----------|
| 폼 클렌저 | 5.5~6.0 | 나이아신아마이드, 알부틴 |
| 젤 클렌저 | 5.0~5.5 | 코직산, 감초추출물 |
| 밀크/크림 클렌저 | 5.5~6.5 | 트라넥삼산, 알부틴 |
| 오일 클렌저 | pH 무관 | 지용성 미백제 |

---

## 실무 권장사항

### 클렌저 개발 시
1. **나이아신아마이드 2~5%** → 가장 안전한 선택
2. **pH 5.5** 타겟 → 피부 친화적 + 성분 안정성
3. **버퍼 시스템** 사용 → 희석 시에도 pH 유지

### 피해야 할 것
- 고농도 비타민C 클렌저 (pH 너무 낮음)
- pH 7.0 이상 클렌저 (장벽 손상)
- 짧은 접촉 시간에 고기대 금물

### 현실적 조언
```
클렌저는 접촉 시간 30초~1분
→ 미백 효과보다 "자극 최소화"가 우선
→ 본격 미백은 세럼/크림에서
```
