## 질문
Alpha Arbutin vs Beta Arbutin: 클렌저 제형에서의 안정성, 효능 차이, 권장 농도를 비교 분석해줘.

## 응답

웹 검색 권한이 없어서 제 지식을 기반으로 분석해줄게.

---

## 1. 구조적 차이

| 항목 | Alpha Arbutin | Beta Arbutin |
|------|--------------|--------------|
| **화학 구조** | α-D-glucopyranoside | β-D-glucopyranoside |
| **원료 출처** | 합성 (효소적 합성) | 천연 추출 (베어베리, 월귤) |
| **가격** | 고가 (10~15배) | 저가 |

---

## 2. 효능 비교

| 항목 | Alpha Arbutin | Beta Arbutin |
|------|--------------|--------------|
| **티로시나제 억제력** | 10배 이상 강함 | 상대적으로 약함 |
| **피부 흡수율** | 높음 | 낮음 |
| **미백 효과** | 빠르고 강력 | 완만하고 온화 |
| **하이드로퀴논 전환** | 거의 없음 | 일부 전환 가능성 |

**핵심**: Alpha Arbutin이 동일 농도에서 훨씬 효과적

---

## 3. 안정성 (클렌저 제형 기준)

| 항목 | Alpha Arbutin | Beta Arbutin |
|------|--------------|--------------|
| **pH 안정 범위** | 3.5~6.5 (넓음) | 4.0~6.0 (좁음) |
| **열 안정성** | 우수 (80°C까지) | 취약 (60°C 이상 분해) |
| **광 안정성** | 우수 | 보통 |
| **계면활성제 호환** | 양호 | 일부 비이온성 권장 |

### 클렌저 제형 시 고려사항

```
문제점:
- 클렌저는 접촉 시간이 짧음 (30초~1분)
- 세정 시 대부분 씻겨나감
- 활성 성분 전달 효율 낮음

권장:
- Alpha Arbutin 선호 (짧은 시간에도 효과)
- pH 5.0~5.5 유지
- 가열 공정 최소화
```

---

## 4. 권장 농도

| 제형 | Alpha Arbutin | Beta Arbutin |
|------|--------------|--------------|
| **클렌저** | 0.5~1.0% | 1.0~2.0% |
| **토너/에센스** | 1.0~2.0% | 2.0~4.0% |
| **크림** | 0.5~2.0% | 1.0~3.0% |
| **최대 권장** | 2.0% | 7.0% |

**클렌저 특이사항**: 씻어내는 제형이라 효과 기대치 낮음 → 마케팅 클레임용으로 활용하거나, leave-on 제품과 병행 권장

---

## 5. 클렌저 제형 결론

| 기준 | 추천 |
|------|------|
| **효능 우선** | Alpha Arbutin 0.5~1.0% |
| **비용 우선** | Beta Arbutin 1.5~2.0% |
| **안정성 우선** | Alpha Arbutin |

### 최종 권장

```
클렌저에서 미백 효과를 기대하기보다는:
1. 브랜드 스토리/마케팅 목적 → Beta Arbutin 1.0% (비용 효율)
2. 실제 효능 목적 → Alpha Arbutin을 세럼/크림에 배합 권장
3. 클렌저에 꼭 넣어야 한다면 → Alpha Arbutin 0.5~1.0%
```

추가로 특정 제형이나 다른 성분과의 배합 궁합이 궁금하면 알려줘.
