## 응답

# 클렌저 pH 최적화: 미백 성분별 전략

## 피부 pH 기본 이해

**건강한 피부 pH**: 4.5-5.5 (약산성)
- 산성 맨틀 유지 → 유해균 억제, 장벽 기능 보호
- pH 7 이상 클렌저 반복 사용 시 → 장벽 손상, 건조, 자극 유발

---

## 미백 성분별 최적 pH 범위

| 성분 | 최적 활성 pH | 클렌저 권장 pH | 비고 |
|------|-------------|---------------|------|
| **비타민 C (L-AA)** | 2.5-3.5 | 3.0-4.0 | pH 3.5 이상에서 급격히 불안정 |
| **나이아신아마이드** | 5.0-7.0 | 5.0-6.0 | 피부 pH와 호환성 우수 |
| **알부틴** | 5.0-7.0 | 5.0-5.5 | 산성에서 하이드로퀴논 분해 위험 |
| **코직산** | 4.5-5.5 | 4.5-5.0 | 피부 pH와 일치, 이상적 |
| **아젤라산** | 4.0-5.0 | 4.5-5.0 | 피부 친화적 |
| **트라넥삼산** | 5.5-7.5 | 5.5-6.0 | 중성 쪽 선호 |
| **감초추출물** | 4.5-6.5 | 5.0-5.5 | 넓은 범위에서 안정 |

---

## 밸런스 전략

### 1. 성분별 제형 선택

**낮은 pH 필요 (3.0-4.5)**: 비타민 C, 코직산
- 워시오프 클렌저보다 **세럼/토너**로 분리 권장
- 클렌저에 넣으면 접촉시간 짧아 효과↓, 자극↑

**피부 pH 호환 (4.5-5.5)**: 나이아신아마이드, 알부틴, 감초
- 클렌저 제형에 **적합**
- 세정 + 미백 동시 가능

### 2. 실무 권장안

```
[아침]
저자극 약산성 클렌저 (pH 5.0-5.5)
  └─ 나이아신아마이드 or 감초추출물 함유

[저녁]
1차: 오일/밤 클렌저 (pH 무관, 계면활성제 최소)
2차: 약산성 폼/젤 (pH 4.5-5.5)
  └─ 알부틴, 코직산 함유 가능
```

### 3. 제형 설계 시 주의점

| 고려사항 | 권장 |
|---------|------|
| 비타민 C | 클렌저 제외, 별도 세럼으로 |
| 나이아신아마이드 | 0.5-2% 농도로 클렌저 적합 |
| 계면활성제 | 아미노산계 (글루타메이트, 글리시네이트) |
| 버퍼 시스템 | 시트르산/시트레이트로 pH 안정화 |
| 접촉시간 | 60초 이내 권장 (자극 최소화) |

---

## 핵심 요약

1. **클렌저 최적 pH**: 4.5-5.5 (피부 pH 존중)
2. **클렌저에 적합한 미백성분**: 나이아신아마이드, 알부틴, 감초
3. **별도 제형 권장**: 비타민 C (세럼), 고농도 코직산
4. **버퍼 시스템 필수**: pH 드리프트 방지
5. **순한 계면활성제**: 아미노산계 선택

원하면 특정 성분 조합이나 제형 예시 더 상세하게 다룰 수 있어.
