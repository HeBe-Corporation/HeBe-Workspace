# 글로벌 클린뷰티 미백 클렌저 포뮬레이션 가이드

## 클린뷰티 필수 요건

### 1. 비건 인증 기준
**동물 유래 성분 대체:**
| 기존 성분 | 비건 대체 성분 |
|-----------|----------------|
| 콜라겐 | 식물성 콜라겐 부스터 (비타민C + 아미노산) |
| 라놀린 | 시어버터, 올리브 스쿠알란 |
| 케라틴 | 밀 단백질, 대두 단백질 |
| 꿀/밀랍 | 아가베 시럽, 칸델릴라 왁스 |
| 실크 | 식물성 글리세린 |

### 2. 크루얼티프리 구현
- **Leaping Bunny / PETA 인증** 취득 권장
- 원료 공급사도 동물실험 미실시 확인 필수
- 중국 수출 시 별도 전략 필요 (중국은 동물실험 요구)

### 3. 마이크로플라스틱 프리
**금지 성분:**
- Polyethylene (PE) 비즈
- Polypropylene (PP)
- Polymethyl methacrylate (PMMA)
- Nylon

**대체 스크럽제:**
- 호두껍질 분말
- 대나무 파우더
- 쌀겨
- 파파야 효소 (물리적 스크럽 없이 각질 제거)

## 비건 미백 클렌저 기본 포뮬레이션

### 추천 처방 (100g 기준)

```
[수상]
정제수                     60-65%
글리세린 (식물성)           3-5%
나이아신아마이드            2-4%     ← 미백 핵심
알파아르부틴               1-2%     ← 미백 보조

[계면활성제 - 마일드]
코코일글루타민산나트륨      8-10%    ← 아미노산계
데실글루코사이드           3-5%     ← 당계

[유상]
스쿠알란 (올리브 유래)      1-2%
호호바오일                 1%

[기능성]
비타민C유도체 (AA2G)        1-2%     ← 미백
감초추출물                 0.5-1%   ← 미백/진정
캐모마일추출물             0.3%     ← 진정

[기타]
잔탄검                     0.2-0.3%
구연산 (pH 조절)           적량 (pH 5.5-6.0 목표)
페녹시에탄올 + 에틸헥실글리세린  0.8-1%  ← 방부
```

## 미백 성분 선택 가이드

### 비건 친화적 미백 성분

| 성분 | 농도 | 특징 |
|------|------|------|
| **나이아신아마이드** | 2-5% | 멜라닌 전달 억제, 자극 적음 |
| **알파아르부틴** | 1-2% | 합성 가능, 티로시나제 억제 |
| **비타민C 유도체** | 1-3% | AA2G, SAP 등 안정형 |
| **감초추출물** | 0.5-1% | 글라브리딘 함유 |
| **트라넥사믹애씨드** | 1-2% | 합성, 미백 효과 우수 |
| **코직산** | 1-2% | 발효 유래 |

### 피해야 할 성분
- 하이드로퀴논 (EU 금지)
- 수은 화합물
- 고농도 레티노이드 (클렌저에 부적합)

## 실무 팁

1. **원료 수급**: 비건 인증된 원료만 사용 (공급사 확인서 필수)
2. **교차오염 방지**: 동물성 원료 사용 라인과 분리
3. **라벨링**: 클린뷰티 클레임은 증빙 가능해야 함
4. **pH 관리**: 미백 성분 안정성을 위해 pH 5.5-6.0 유지
5. **안정성 테스트**: 최소 3개월 가속 시험

추가로 특정 타겟 시장(EU/US/아시아)이나 제형(폼/젤/밀크)에 맞춘 상세 처방이 필요하면 알려줘.
