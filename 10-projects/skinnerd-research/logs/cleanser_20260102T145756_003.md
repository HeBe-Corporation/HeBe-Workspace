# 클렌저 리서치 #3
- 시간: 2026. 1. 3. 오전 12:59:07
- 질문: 3/35

## 질문
Alpha Arbutin vs Beta Arbutin: 클렌저 제형에서의 안정성, 효능 차이, 권장 농도를 비교 분석해줘.

## 응답

웹 검색 권한이 없어서 내 지식 기반으로 비교 분석할게.

---

## 1. 구조적 차이

| 구분 | Alpha Arbutin | Beta Arbutin |
|------|---------------|--------------|
| **화학 구조** | α-glucoside 결합 | β-glucoside 결합 |
| **원료** | 합성 (효소적 합성) | 천연 추출 (베어베리, 월귤나무) |
| **가격** | 고가 | 상대적 저가 |

---

## 2. 안정성 비교 (클렌저 제형)

| 요소 | Alpha Arbutin | Beta Arbutin |
|------|---------------|--------------|
| **pH 안정성** | pH 3.5-6.5에서 안정 | pH 5-7에서 최적 |
| **열 안정성** | 우수 (60°C까지) | 보통 (열에 취약) |
| **가수분해** | 낮음 | 높음 (하이드로퀴논 전환 위험) |
| **클렌저 적합성** | ⭐⭐⭐⭐ | ⭐⭐ |

**클렌저 제형 핵심 이슈:**
- 클렌저는 물 함량 높고, 피부 접촉 시간 짧음 (30초-2분)
- Beta Arbutin은 수성 환경에서 **가수분해되어 하이드로퀴논으로 전환**될 수 있음
- Alpha Arbutin이 클렌저에 더 적합

---

## 3. 효능 비교

| 지표 | Alpha Arbutin | Beta Arbutin |
|------|---------------|--------------|
| **타이로시나제 억제력** | 10배 이상 강함 | 기준 (1x) |
| **미백 효과** | 빠름/강력 | 온화/점진적 |
| **피부 자극** | 낮음 | 매우 낮음 |
| **침투력** | 높음 | 보통 |

---

## 4. 권장 농도

### Alpha Arbutin
| 제형 | 권장 농도 | 비고 |
|------|----------|------|
| 클렌저 | **0.5-1%** | 접촉 시간 짧아 고농도 가능 |
| 세럼/에센스 | 1-2% | 일반적 사용 |
| 크림 | 0.5-1% | |

### Beta Arbutin
| 제형 | 권장 농도 | 비고 |
|------|----------|------|
| 클렌저 | **1-2%** | 효능 낮아 더 높은 농도 필요 |
| 세럼/에센스 | 2-5% | |
| 크림 | 1-3% | |

---

## 5. 클렌저 제형 결론

### 추천: **Alpha Arbutin**

| 이유 | 설명 |
|------|------|
| ✅ 안정성 | 수성 환경, 계면활성제와 호환 우수 |
| ✅ 효율성 | 낮은 농도로 동등 효과 |
| ✅ 안전성 | 가수분해 위험 낮음 |
| ⚠️ 단점 | 원가 높음 |

### 제형 팁
```
권장 pH: 4.5-6.0
권장 농도: Alpha 0.5-1% / Beta 사용 비추천
첨가 시점: 냉각 후 (45°C 이하)
호환 성분: 나이아신아마이드, 비타민 C 유도체
비호환: 강산성/강알칼리 환경
```

| 안정성 | 우수 | 취약 |
