# Glutathione 클렌저 제형 분석

## 1. 최적 농도

| 용도 | 농도 범위 | 비고 |
|------|-----------|------|
| **일반 클렌저** | 0.1-0.5% | 일상 사용, 자극 최소화 |
| **미백 집중** | 0.5-2.0% | 효능 중심, 전문가용 |
| **권장 최적** | **0.5-1.0%** | 효능/안정성 균형점 |

> 클렌저는 wash-off 제품이라 2% 이상은 비용 대비 효과 낮음

---

## 2. 안정화 방법

### 핵심 문제
Glutathione(GSH)은 **tripeptide(Glu-Cys-Gly)** 구조로 산화에 매우 취약

### 안정화 전략

| 방법 | 설명 |
|------|------|
| **pH 조절** | pH 4.0-6.0 유지 (중성~알칼리에서 급격 분해) |
| **산화방지제 병용** | Ascorbic acid, Sodium metabisulfite, BHT |
| **킬레이트제** | EDTA, Phytic acid (금속이온 제거) |
| **캡슐화** | Liposome, Cyclodextrin 포접 |
| **환원형 유지** | L-Glutathione Reduced (GSH) 사용 |
| **차광 용기** | UV 차단 (광분해 방지) |
| **질소 충전** | 산소 접촉 최소화 |

### 추천 조합
```
Glutathione 0.5%
+ Sodium Ascorbyl Phosphate 0.5%
+ EDTA 0.1%
+ pH 5.0-5.5 조정
```

---

## 3. pH 조건

| pH | 안정성 | 피부 적합성 | 판정 |
|----|--------|------------|------|
| < 3.5 | 높음 | 자극 우려 | ✗ |
| **4.0-5.5** | 양호 | 피부 친화적 | ✓ 최적 |
| 6.0-7.0 | 불안정 | 양호 | △ |
| > 7.5 | 급속 분해 | - | ✗ |

**권장 pH: 4.5-5.5** (피부 산성막과 유사, 안정성 확보)

---

## 4. 클렌저에서의 효능 발현 메커니즘

### 4.1 미백 메커니즘

```
Glutathione
    ↓
1) Tyrosinase 저해 → Melanin 합성 억제
2) Pheomelanin 전환 촉진 (어두운 Eumelanin → 밝은 Pheomelanin)
3) 항산화 → ROS 제거 → 멜라닌 과생성 방지
```

### 4.2 클렌저 제형의 한계와 극복

| 한계 | 이유 | 극복 방안 |
|------|------|-----------|
| 짧은 접촉 시간 | 30초-1분 세정 | 고농도 사용, 마사지 시간 확보 |
| 계면활성제 간섭 | 미셀 형성으로 침투 방해 | Mild surfactant 선택 |
| 즉각 효과 미미 | 누적 효과 필요 | Leave-on 제품 연계 |

### 4.3 실제 기대 효과

**클렌저 단독으로는 미백 효과 제한적**

주요 역할:
1. **Priming 효과** - 후속 미백 에센스 침투 향상
2. **항산화 보조** - 피부 표면 산화 스트레스 감소
3. **지속 사용 시 톤 균일화** - 각질층 케어
4. **Brightening ingredient 전달** - 세안 중 단기 노출

## 5. 제형 설계 권장사항

```yaml
타입: Low-foam gel cleanser
pH: 5.0 ± 0.5
Glutathione: 0.5-1.0%
보조 성분:
  - Niacinamide 2% (미백 시너지)
  - Sodium Ascorbyl Phosphate 0.5% (안정화 + 미백)
  - EDTA 0.1%
계면활성제: Cocamidopropyl Betaine (자극↓)
용기: Airless pump, 차광
보관: 냉암소 권장
```
