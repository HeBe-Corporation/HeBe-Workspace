# 클렌저 OEM 품질 관리 체크포인트

## 1. 미생물 검사

| 항목 | 기준 | 검사 시점 |
|------|------|----------|
| 총 호기성 생균수 | ≤1,000 CFU/g (일반), ≤500 CFU/g (영유아용) | 벌크, 완제품 |
| 진균(곰팡이/효모) | ≤100 CFU/g | 벌크, 완제품 |
| 대장균(E.coli) | 불검출 | 완제품 |
| 녹농균(P.aeruginosa) | 불검출 | 완제품 |
| 황색포도상구균(S.aureus) | 불검출 | 완제품 |

**방부력 테스트 (Challenge Test)**
- 접종균: E.coli, P.aeruginosa, S.aureus, C.albicans, A.brasiliensis
- 판정: 28일간 균 감소율 확인

## 2. 안정성 테스트

### 물리적 안정성
| 항목 | 조건 | 기간 |
|------|------|------|
| 가속 안정성 | 40±2°C, 75±5% RH | 3~6개월 |
| 장기 안정성 | 25±2°C, 60±5% RH | 12~24개월 |
| 저온 안정성 | 4°C | 3개월 |
| 온도 순환 | -10°C ↔ 45°C 반복 | 6사이클 |
| 광안정성 | 형광등/자외선 노출 | 1~3개월 |

### 확인 항목
- **외관**: 색상 변화, 분리, 침전, 결정화
- **pH**: 초기 대비 ±0.5 이내
- **점도**: 초기 대비 ±15% 이내
- **향**: 변향 여부
- **유화 안정성**: 원심분리 3,000rpm/30분 후 분리 여부

## 3. 사용감(관능) 테스트

### 세정력
| 항목 | 측정 방법 |
|------|----------|
| 피지 제거력 | 인공피지 도포 후 세정률 (%) |
| 메이크업 제거력 | 표준 메이크업 후 세정률 |
| 거품력 | Ross-Miles법, 실린더 테스트 |
| 거품 지속성 | 5분 후 잔존 거품량 |

### 피부 자극
| 항목 | 기준 |
|------|------|
| 첩포 테스트 | ICDRG 기준 (48hr, 72hr 판독) |
| 누적 자극 테스트 | 21일 반복 도포 |
| 안자극 테스트 (아이 제품) | Draize법 또는 대체시험 |
| 경피수분손실(TEWL) | 세정 전후 비교 |
| 피부 pH 변화 | 세정 후 30분 내 회복 여부 |

### 소비자 사용감
| 평가 항목 | 척도 |
|-----------|------|
| 거품 질감 | 조밀함, 탄력, 크리미함 |
| 세정 후 당김 | 1~5점 |
| 세정 후 촉촉함 | 1~5점 |
| 헹굼성 | 잔여감 유무 |
| 향 선호도 | 1~5점 |

---

## 4. 이화학 검사

| 항목 | 방법 |
|------|------|
| pH | pH meter (25°C) |
| 점도 | 점도계 (Brookfield 등) |
| 비중 | 비중계/피크노미터 |
| 유효 성분 함량 | HPLC, GC 등 |
| 중금속 | 납 ≤20ppm, 비소 ≤10ppm, 수은 ≤1ppm, 안티몬 ≤10ppm |

---

## 5. 검사 타이밍 요약

```
원료 입고 → 원료 COA 확인, 미생물/이화학 스팟체크
    ↓
벌크 제조 → pH, 점도, 외관, 미생물
    ↓
충전 → 중량, 기밀성, 외관
    ↓
완제품 → 전 항목 검사 + 안정성 샘플 채취
    ↓
출하 → 출하 검사 성적서 발행
```
