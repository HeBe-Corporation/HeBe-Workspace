## 응답

# 동남아 고온고습 환경에서 클렌저 텍스처 안정성 확보

## 기후 조건
- **온도**: 25~40°C (창고/운송 중 최대 50°C)
- **습도**: 70~95% RH
- **도전 과제**: 점도 저하, 상 분리, 미생물 오염

## 안정성 확보 전략

### 1. 점증제 선택
```
권장:
- Carbomer 940/980 (0.2~0.5%) — 열 안정성 우수
- Acrylates/C10-30 Alkyl Acrylate Crosspolymer — 전해질 내성
- Xanthan Gum + Cellulose Gum 복합 — 시너지 효과
- Hydroxypropyl Starch Phosphate — 천연 대안

피할 것:
- 저분자 HEC 단독 사용 — 고온에서 급격한 점도 저하
```

### 2. 유화 안정화 (O/W 크림 타입)
- **HLB 밸런스**: 11~13 유지
- **공유화제** 추가: Cetearyl Alcohol + Ceteareth-20 조합
- **상분리 방지**: Stearic Acid 1~2% 첨가

### 3. 방부 시스템 강화
```
고온고습 = 미생물 리스크 ↑

권장 조합:
- Phenoxyethanol 0.8% + Ethylhexylglycerin 0.3%
- 또는 Phenoxyethanol + Caprylyl Glycol + Chlorphenesin
- pH 4.5~5.5 유지 (방부 효과 극대화)
```

### 4. 패키징
- **에어리스 펌프**: 공기/습기 유입 차단
- **알루미늄 튜브**: 차광 + 수분 차단
- **실리카겔 캡**: 튜브 마개 내 습기 흡수

## 실무 팁

1. **과점도 처방**: 초기 점도를 목표보다 **15~20% 높게** 설정
2. **현지 파일럿**: 베트남/캄보디아 현지 창고에서 실제 보관 테스트 (3개월)
3. **배치 관리**: 여름 생산분은 점증제 함량 소폭 증가 고려
4. **유통기한**: 열대 기후용은 18~24개월 권장 (일반 36개월 대비 단축)
