```

→ 클렌저의 본질적 기능 완벽 수행
→ 피지, 메이크업, 선크림 깔끔히 제거
→ "세정력 당연히 좋고, 그 위에 미백/쿨링까지"
```

```
Niacinamide (2.5%) + Alpha-Arbutin (0.8%) +
Tranexamic Acid (1.5%) + 3-O-Ethyl Ascorbic Acid (1.5%)

```

**USP #3: 열대 기후 쿨링 (TROPICAL COOLING)**
```
Menthoxypropanediol 0.3% + Cucumber Extract 0.5%

→ 동남아 40°C/80% 습도 환경 최적화
→ 세안 시 상쾌한 쿨링감
→ 멘톨보다 저자극, 지속적인 시원함
```

**USP #4: K-Beauty 핫 성분 PDRN**

```
→ 2025-2026 K-Beauty #1 트렌드 성분
```

**USP #5: pH 5.5 저자극 포뮬러**

## 6. SWOT 분석 (Skin Nerd)

### Strengths (강점)
1. 강력한 세정력 (SCI+CAPB 14% 시스템)
2. 4종 미백 성분 콤플렉스
3. 열대 기후 쿨링 (멘톡시프로판디올+오이)
4. K-Beauty 트렌드 성분 (PDRN, 발효)
5. 아렌시아 약점 보완 (자극, 향, 패키징)
6. 동남아 타겟 최적화 (기후, 규제, 쿨링)
7. 무향/미세향 옵션으로 민감성 대응

### Weaknesses (약점)
1. 신규 브랜드 인지도 부족
2. 아렌시아의 "떡" 텍스처 바이럴 대비 부족
3. 초기 마케팅 비용 부담
4. OEM 품질 관리 리스크

### Opportunities (기회)
1. 동남아 스킨케어 시장 성장 (CAGR 5-10%)
2. 미백/Glow 클렌저 틈새 시장
3. K-Beauty 글로벌 인기
4. Whipped/Cotton Candy 텍스처 트렌드
5. PDRN 성분 바이럴 가능성

### Threats (위협)
1. 아렌시아 후속 제품 출시
2. 대기업 유사 제품 진입
3. 원료 가격 상승 리스크
4. 동남아 규제 변화 가능성
