## 2. 동료가 복사해서 사용하는 방법

동료에게 아래 가이드를 공유하면 돼:

### 📋 HeBe-Workspace 복사 가이드

#### 방법 A: Git Clone (추천)
```bash
# 1. 원하는 폴더로 이동
cd C:\Users\{사용자이름}\.claude

# 2. 저장소 복제
git clone https://github.com/mikaelkyujinkong-ship-it/HeBe-Workspace.git

# 3. 폴더 열기
cd HeBe-Workspace
```

#### 방법 B: ZIP 다운로드
1. https://github.com/mikaelkyujinkong-ship-it/HeBe-Workspace 접속
2. 초록색 **Code** 버튼 클릭
3. **Download ZIP** 선택
4. 압축 해제 후 `C:\Users\{사용자이름}\.claude\` 폴더에 이동

#### 사용 시작
1. **VSCode** 또는 **Claude Code**에서 해당 폴더 열기
2. `/setup-workspace` 실행해서 자신만의 `CLAUDE.md` 생성
3. 커맨드 사용: `/daily-note`, `/todos`, `/thinking-partner` 등

#### 주의사항
- `.env` 파일은 복사되지 않음 → 본인 API 키 직접 설정 필요
- `output/` 폴더의 데이터는 예시용 → 삭제해도 됨
