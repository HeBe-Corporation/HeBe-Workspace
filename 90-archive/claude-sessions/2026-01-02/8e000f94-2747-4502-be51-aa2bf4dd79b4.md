## 🤖 Claude

저는 Claude Code (CLI 도구)이기 때문에 사용자의 구독 상태나 계정 정보에 접근할 수 없습니다.

구독 상태를 확인하려면:
1. **claude.ai** 웹사이트에 로그인
2. 설정(Settings) → 구독(Subscription) 메뉴 확인

또는 Claude Code를 사용 중이시라면, 정상적으로 작동하고 있다면 유효한 인증이 되어 있는 것입니다.

## 🤖 Claude

`claude logout`은 터미널에서 직접 실행하는 CLI 명령어입니다.

터미널에서 다음을 입력하세요:

```bash

```

이 명령어를 실행하면 현재 세션에서 로그아웃됩니다.

## 🤖 Claude

현재 저와 대화 중인 Claude Code 세션 안에 계십니다.

로그아웃하려면:

1. 먼저 이 세션을 종료하세요: `/exit` 입력 또는 `Ctrl+C`
2. 그 다음 터미널에서 `claude logout` 실행

또는 세션 내에서 `/logout` 명령어를 사용할 수도 있습니다.
