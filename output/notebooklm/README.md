# NotebookLM용 파일 가이드

## 파일 목록

| 파일 | 용도 | 크기 |
|------|------|------|
| `gash-review-report.md` | 분석 리포트 (필수) | 9KB |
| `gash-reviews-sample.txt` | 원본 리뷰 샘플 100개 | ~50KB |

---

## NotebookLM 업로드 방법

### Step 1: NotebookLM 접속
1. https://notebooklm.google.com 접속
2. "새 노트북 만들기" 클릭

### Step 2: 파일 업로드
1. "소스 추가" 클릭
2. `gash-review-report.md` 업로드 (필수)
3. (선택) `gash-reviews-sample.txt` 업로드

### Step 3: 활용

#### 기본 질문 예시
- "이 제품의 핵심 강점 3가지는?"
- "고객 불만 TOP 5를 알려줘"
- "마케팅 메시지 추천해줘"
- "30대 이상 고객을 위한 개선점은?"

#### PPT용 질문 예시
- "5장짜리 PPT 구성 제안해줘"
- "투자자에게 보여줄 핵심 지표는?"
- "경쟁사 대비 강점을 한 문장으로"

## PPT 제작 가이드

### 추천 구성 (5-7장)

| 슬라이드 | 내용 | NotebookLM 질문 |
|----------|------|-----------------|
| 1 | 타이틀 + 핵심 수치 | "한 줄 요약과 핵심 지표" |
| 2 | 평점 분포 차트 | "평점 분포 데이터" |
| 3 | 옵션별 인기도 | "옵션별 판매 비율" |
| 4 | 긍정 키워드 워드클라우드 | "가장 많이 언급된 긍정 키워드 10개" |
| 5 | 고객 불만 & 개선점 | "불만 유형과 해결 방안" |
| 6 | 액션 아이템 | "마케팅 메시지 제안" |
| 7 | 요약 & 결론 | "최종 결론 한 문장" |

### NotebookLM → PPT 워크플로우

1. **NotebookLM에서 질문** → 답변 복사
2. **Gamma.app** 또는 **Canva AI**에 붙여넣기
3. 자동 PPT 생성
4. 디자인 조정

### 추천 도구

| 도구 | 특징 | 링크 |
|------|------|------|
| Gamma.app | 텍스트 → PPT 자동 생성 | gamma.app |
| Canva AI | 템플릿 기반 PPT | canva.com |
| Beautiful.ai | 스마트 레이아웃 | beautiful.ai |

## 팁
