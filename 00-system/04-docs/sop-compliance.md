# 위생허가(컴플라이언스) SOP

> 허가/인증 관리, 만료/갱신 태스크, 증빙 파일 관리

---

## 프로세스 단계 (표준)

```
1. 신규/변경 필요 발생 (신규 매장/품목/설비)
   ↓
2. 요구서류 리스트업 (국가/기관/업종별)
   ↓
3. 서류 수집/작성 (번역 포함)
   ↓
4. 제출 (대행사/직접)
   ↓
5. 보완 요청 대응
   ↓
6. 현장 점검 (필요시)
   ↓
7. 승인/발급
   ↓
8. 파일 보관 (Drive) + Notion 레코드 업데이트
   ↓
9. 갱신 태스크 자동 생성 (만료 90일 전)
```

---

## 체크리스트 템플릿

### 공통 서류
- [ ] 사업자/법인 기본서류 (등록증 등)
- [ ] 임대차/주소 증빙
- [ ] 시설/도면/장비 리스트
- [ ] 위생/안전 교육 이수 (해당 시)
- [ ] 제품 성분/라벨 (해당 시)
- [ ] 시험성적서/검사 (해당 시)
- [ ] 수수료 납부 영수증
- [ ] 결과 통지서/허가증 스캔본

## 알림 규칙

expiryDate 기준으로 단계별 알림:

| D-day | 액션 |
|-------|------|
| **D-90** | 준비 시작 (renewalStartDate) |
| **D-60** | 서류 1차 완료 목표 |
| **D-30** | 제출 완료 목표 |
| **D-14** | 리스크 경고 (미완료면 P0) |
| **D-7** | 임원/매니저 escalation |

## 리스크 스코어

### High
- 영업 중단 가능
- 벌금/몰수 리스크
- 수입/수출 통관에 영향

### Medium
- 제품 판매 제한

### Low
- 내부 문서성/권고사항
