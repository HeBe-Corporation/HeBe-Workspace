# Sample Data for Claude Code Practice

Claude Code 데이터 분석 실습용 샘플 데이터입니다.

## 데이터셋

| 파일명 | 설명 | 추천 실습 |
|--------|------|----------|
| `sample_sales_data.csv` | 판매 데이터 (제품, 지역, 영업사원별) | 매출 분석 |
| `customer_data.csv` | 고객 정보 (구매 이력, LTV) | 고객 세분화 |
| `employee_survey_data.csv` | 직원 만족도 설문 | 설문 분석 |
| `marketing_campaign_data.csv` | 마케팅 캠페인 성과 | ROI 분석 |
| `inventory_data.csv` | 재고 현황 | 재고 관리 |
