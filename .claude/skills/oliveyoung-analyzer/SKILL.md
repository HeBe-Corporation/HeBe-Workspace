# 올리브영 리뷰 분석기

올리브영 상품의 리뷰를 크롤링하고 신제품 개발 인사이트를 도출합니다.

## 트리거 키워드
- "올리브영 분석", "올영 리뷰", "올리브영 크롤링"
- "화장품 리뷰 분석", "뷰티 제품 분석"
- oliveyoung.co.kr URL 포함

## 입력 형식
1. **올리브영 URL**: `https://www.oliveyoung.co.kr/store/goods/getGoodsDetail.do?goodsNo=A000000235842`
2. **상품번호만**: `A000000235842` 또는 `000000235842`
3. **자연어**: "올리브영 가쉬 버블팩 리뷰 분석해줘"

## 워크플로우

### Step 1: 입력 확인
사용자 입력에서 올리브영 URL 또는 상품번호 추출

### Step 2: 크롤링 실행
```bash
npx ts-node scripts/oliveyoung-analyzer.ts <URL 또는 상품번호>
```

### Step 3: 분석 결과 확인
`./output/` 폴더에 생성된 파일 확인:
- `oliveyoung-{상품번호}-{날짜}.json` - 원본 데이터
- `oliveyoung-{상품번호}-{날짜}.csv` - CSV 형식
- `{상품명}-review-analysis-{날짜}.md` - 분석 리포트

### Step 4: AI 인사이트 제공
분석 리포트를 읽고 추가 인사이트 제공:
- SWOT 분석 심화
- 경쟁사 대비 포지셔닝
- 신제품 컨셉 제안
- 마케팅 메시지 추천

## 분석 항목

### 기본 분석 (자동)
- 평점 분포 및 평균
- 옵션별 만족도
- 피부타입별 분포
- 재구매율, 사진 포함율

### 키워드 분석 (자동)
- 긍정 키워드 TOP 15 (촉촉, 수분, 탄력 등)
- 부정 키워드 TOP 15 (자극, 트러블, 비싸 등)
- 카테고리별 그룹핑

### 개선 기회 분석 (자동)
- 불만 유형 → 신제품 기회 매핑
- 실제 리뷰 샘플 포함
- 제안 액션 아이템

### AI 심화 분석 (Claude)
- 고객 세그먼트별 니즈 분석
- 프리미엄화 가능성
- 마케팅 카피 추천
